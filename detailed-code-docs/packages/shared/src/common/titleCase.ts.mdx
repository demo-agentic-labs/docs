## High-level description
This code defines a utility function `toTitleCase` that converts a given string to title case format. It uses the `startCase` function from the Lodash library to achieve this transformation.

## Symbols

### `toTitleCase`
#### Description
The `toTitleCase` function takes a string input, trims any leading or trailing whitespace, converts it to lowercase, and then applies the `startCase` function from Lodash to capitalize the first letter of each word.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| str | string | The input string to be converted to title case |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | string | The input string converted to title case |

#### Internal Logic
1. Trim any leading or trailing whitespace from the input string using `str.trim()`.
2. Convert the trimmed string to lowercase using `trimmedStr.toLowerCase()`.
3. Apply the `startCase` function from Lodash to capitalize the first letter of each word.
4. Return the resulting title-cased string.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| lodash | The `startCase` function is imported from Lodash to handle the capitalization of words |

## Notes
- The function ensures consistent capitalization by first converting the entire string to lowercase before applying the title case transformation.
- By using `trim()`, the function handles inputs with leading or trailing whitespace gracefully.
- The use of Lodash's `startCase` function provides a robust way to handle various edge cases in title casing, such as handling apostrophes, hyphens, and other special characters.
