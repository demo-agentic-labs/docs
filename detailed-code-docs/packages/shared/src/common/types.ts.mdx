## High-level description
This file defines TypeScript types and utility functions for handling boolean conversions and ensuring at least one property is required in an object type. It provides a way to convert strings to booleans and defines a type utility for creating object types with at least one required property.

## Symbols

### `AtLeastOne&lt;T&gt;`
#### Description
A type utility that creates a new type from `T` where at least one property is required.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| T | object type | The original object type |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| AtLeastOne&lt;T&gt; | object type | A new type where at least one property of T is required |

#### Internal Logic
This type uses mapped types and conditional types to create a union of all possible combinations of T where at least one property is required.

### `stringToBoolean`
#### Description
Converts a string to a boolean value, returning undefined if the input is falsy.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string \| undefined | The string to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | boolean \| undefined | The converted boolean value or undefined |

#### Internal Logic
If the input is falsy, it returns undefined. Otherwise, it calls `stringToBooleanRequired`.

### `stringToBooleanRequired`
#### Description
Converts a string to a boolean value. This function is deprecated and should be replaced with `isTrue()`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | The string to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | boolean | The converted boolean value |

#### Internal Logic
Converts the input string to lowercase, trims it, and compares it to "true".
