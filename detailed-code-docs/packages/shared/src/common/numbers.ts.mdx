## High-level description
This code provides utility functions for handling and formatting numbers in TypeScript. It includes functions to convert string values to floats, format numbers to two decimal places, and generate random integers.

## Symbols

### `getFloatValue`
#### Description
Converts a string or number input to a float value.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string \| number | The input value to be converted to a float |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | number | The float value of the input |

#### Internal Logic
- If the input is a string, it uses `parseFloat` to convert it to a number.
- If the input is already a number, it returns it as-is.

### `formatNumber`
#### Description
Truncates a number to two decimal places.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| num | number | The number to be formatted |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | number | The formatted number with two decimal places |

#### Internal Logic
- Multiplies the input by 100.
- Uses `Math.floor` to round down to the nearest integer.
- Divides the result by 100 to get two decimal places.

### `randomInt`
#### Description
Generates a random integer with a specified maximum number of digits.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| maxDigits | number | The maximum number of digits for the random integer (default: 2) |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | number | A random integer with the specified maximum number of digits |

#### Internal Logic
- Uses `Math.random()` to generate a random number between 0 and 1.
- Multiplies the result by 10 raised to the power of `maxDigits`.
- Uses `Math.floor` to round down to the nearest integer.
