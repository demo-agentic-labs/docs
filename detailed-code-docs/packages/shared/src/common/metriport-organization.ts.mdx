## High-level description
This file defines a constant `metriportOrganization` of type `Organization` from the FHIR (Fast Healthcare Interoperability Resources) standard. It contains basic information about the Metriport organization, including its name, address, and contact details.

## Symbols

### `metriportOrganization`
#### Description
A constant that represents the Metriport organization using the FHIR Organization resource type. It includes the organization's name, address, and contact information.

#### Inputs
This is a constant, so it doesn't have inputs.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|

#### Internal Logic
The `metriportOrganization` constant is structured according to the FHIR Organization resource type with the following properties:
1. `resourceType`: Set to "Organization"
2. `name`: Set to "Metriport"
3. `address`: An array containing one address object with street, city, state, postal code, and country
4. `telecom`: An array containing one telecom object with a phone number
