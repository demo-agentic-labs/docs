## High-level description
This code defines utility functions and types for mocking Axios in Jest tests and creating mock Axios responses. It also includes a complex error object structure, likely used for testing error handling scenarios in network requests.

## Code Structure
The code is organized into three main parts: Axios mocking functions, a response creation utility, and a detailed error object structure.

## Symbols

### `mockAxios()`
#### Description
This function creates a mock implementation of Axios and its main HTTP methods (get, post, put, delete) for use in Jest tests.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mockedAxios | Object | An object containing mocked Axios instance and HTTP methods |

#### Internal Logic
1. Uses Jest's `mocked` function to create a mock of the Axios library.
2. Creates mock implementations for Axios instance and its HTTP methods.
3. Returns an object with these mocked functions.
