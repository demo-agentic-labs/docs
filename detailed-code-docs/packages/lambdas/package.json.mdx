### Scripts
#### Description
Defines various npm scripts for cleaning, building, testing, and linting the project.

#### Internal Logic
- `clean`: Removes dist and node_modules directories, and cleans the playwright layer.
- `build`: Compiles TypeScript files using a specific tsconfig file.
- `build:cloud`: Compiles TypeScript files using a cloud-specific tsconfig file.
- `typecheck`: Runs TypeScript compiler without emitting files to check for type errors.
- `prep-deploy`: Runs a deployment preparation script.
- `lint`: Runs ESLint on TypeScript files.
- `test`: Runs Jest tests with specific configurations.
