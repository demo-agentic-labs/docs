## High-level description
This file defines TypeScript type declarations for the "saxon-js" module. It provides type definitions for two main functions: `getResource` and `transform`, along with their associated option types. These declarations enable TypeScript to provide type checking and autocompletion for the Saxon-JS library, which is likely used for XML processing and XSLT transformations.

## Symbols

### `GetResourceOptions`
#### Description
This type defines the options for the `getResource` function. It combines a type property with one of three possible source specifications: location, file, or text.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| type | "text" \| "json" \| "xml" | Specifies the type of resource |
| encoding | "utf8" \| "ucs2" \| "utf16le" \| "latin1" \| "ascii" | Optional encoding for the resource |
| location | string | URL or path to the resource (mutually exclusive with file and text) |
| file | string | File path to the resource (mutually exclusive with location and text) |
| text | string | Direct text content of the resource (mutually exclusive with location and file) |

### `getResource`
#### Description
This function is used to retrieve a resource, likely an XML, JSON, or text file.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | GetResourceOptions | Options specifying the resource to retrieve |
| processing | "async" | Optional parameter to specify asynchronous processing |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | Promise&lt;unknown&gt; | A promise that resolves to the retrieved resource |

### `TransformOptions`
#### Description
This type defines the options for the `transform` function, specifying various parameters for XSLT transformation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| stylesheetText | string | The XSLT stylesheet as a string |
| stylesheetParams | Record&lt;string, any&gt; | Optional parameters for the stylesheet |
| sourceText | string | Optional source XML as a string |
| sourceNode | any | Optional source XML as a node |
| sourceFileName | string | Optional source XML file name |
| sourceBaseURI | string | Optional base URI for the source |
| destination | string | Optional destination for the transformation result |
| initialTemplate | string | Optional name of the initial template to execute |
| initialMode | string | Optional initial mode for the transformation |
| initialFunction | string | Optional initial function to call |

### `transform`
#### Description
This function performs an XSLT transformation using the specified options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| options | TransformOptions | Options for the transformation |
| processing | "async" | Optional parameter to specify asynchronous processing |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | Promise&lt;\{ principalResult: string \}&gt; | A promise that resolves to an object containing the transformation result |
