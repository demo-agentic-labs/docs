## High-level description
This code defines a React component called `Success` that renders a success page for a connection widget. It displays a success message with a green checkmark icon and a "Continue" button. When the button is clicked, it triggers an analytics event and redirects the user to either a custom URL or the main page.

## Code Structure
The `Success` component is the main export of this file. It uses various Chakra UI components for styling and layout. The component also utilizes React Router hooks for navigation and URL parameter handling.

## Symbols

### Success
#### Description
A functional React component that renders the success page of the connection widget.

#### Inputs
This component doesn't accept any props.

#### Internal Logic
1. Retrieves search parameters and navigation function using React Router hooks.
2. Defines a `handleClick` function that:
   - Emits a "connect success" analytics event.
   - Checks for a custom redirect URL in localStorage.
   - Redirects to the custom URL if available, otherwise redirects to the main page.
3. Renders a layout with a success message, checkmark icon, and a "Continue" button.

#### Side Effects
- Emits an analytics event when the "Continue" button is clicked.
- Redirects the user to either a custom URL or the main page.
