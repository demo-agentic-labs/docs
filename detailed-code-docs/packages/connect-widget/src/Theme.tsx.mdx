## High-level description
This code defines a custom theme for a React application using Chakra UI. It extends the default Chakra UI theme to set a global background color that changes based on the light or dark mode of the application.

## Symbols

### `theme`
#### Description
The `theme` constant is created using the `extendTheme` function from Chakra UI. It defines a custom theme that sets a global background color for the body element, which changes based on the color mode (light or dark).

#### Internal Logic
1. The theme uses the `styles.global` property to define global styles.
2. It sets the `body.bg` (background) property using the `mode` function from Chakra UI's theme tools.
3. The `mode` function takes two arguments: the light mode value (`light`) and the dark mode value (`dark`).
4. The background color will be `"gray.100"` in light mode and `"gray.900"` in dark mode.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @chakra-ui/react | Provides the `extendTheme` function for creating custom themes |
| @chakra-ui/theme-tools | Provides the `mode` function for creating responsive color modes |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| dark | string | "gray.900" | The background color used in dark mode |
| light | string | "gray.100" | The background color used in light mode |
