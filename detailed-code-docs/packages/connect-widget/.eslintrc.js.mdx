## High-level description
This file (.eslintrc.js) configures ESLint for the connect-widget package. It sets up the linting rules and parser for TypeScript in a React project, ensuring code quality and consistency.

## Code Structure
The file exports a single configuration object that defines the ESLint setup for the project.

## Symbols

### module.exports
#### Description
This is the main configuration object for ESLint in the connect-widget package. It extends recommended configurations, sets up the TypeScript parser, and defines plugins to be used.

#### Inputs
N/A

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Configuration Object | Object | ESLint configuration settings |

#### Internal Logic
The configuration object includes the following key settings:
1. Extends recommended ESLint configurations and Prettier.
2. Specifies the TypeScript parser.
3. Adds the TypeScript ESLint plugin.
4. Sets the root property to true, indicating this is the top-level configuration file.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @typescript-eslint/parser | Allows ESLint to parse TypeScript code |
| @typescript-eslint/eslint-plugin | Provides TypeScript-specific linting rules |
| eslint-config-prettier | Turns off ESLint rules that might conflict with Prettier |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| extends | Array | ['eslint:recommended', 'plugin:@typescript-eslint/recommended', 'prettier'] | Base configurations to extend |
| parser | String | '@typescript-eslint/parser' | Parser to use for TypeScript files |
| plugins | Array | ['@typescript-eslint'] | ESLint plugins to use |
| root | Boolean | true | Indicates this is the root configuration file |

## References
This configuration file is referenced in the `package.json` file of the connect-widget package, specifically in the `lint` and `lint-fix` scripts:

```json
"scripts": {
  "lint": "npx eslint src/**/*.ts src/**/*.tsx",
  "lint-fix": "npm run lint --fix",
}
```

These scripts use the ESLint configuration defined in this file to lint the TypeScript and TSX files in the `src` directory.
