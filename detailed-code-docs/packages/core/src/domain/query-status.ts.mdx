## High-level description
This code defines types and constants related to query status and progress in a TypeScript module. It provides a set of possible query statuses and a structure for representing query progress, which includes the status and an optional start time.

## Symbols

### `queryStatus`
#### Description
A constant array that defines the possible states of a query.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| queryStatus | readonly ["processing", "completed", "failed"] | An array of string literals representing the possible query statuses |

### `QueryStatus`
#### Description
A type alias that represents the possible values for a query's status.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| QueryStatus | "processing" \| "completed" \| "failed" | A union type of the possible query status values |

### `QueryProgress`
#### Description
An interface that defines the structure for representing the progress of a query.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| QueryProgress | \{ status: QueryStatus; startedAt?: Date \} | An object type representing query progress |
