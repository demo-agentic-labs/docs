## High-level description
This code defines types and constants related to document conversion in a medical data processing system. It specifies valid conversion types (HTML and PDF) and defines input and output structures for the conversion process.

## Symbols

### `validConversionTypes`
#### Description
A constant array that defines the valid conversion types for documents.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| validConversionTypes | readonly ["html", "pdf"] | An array of strings representing valid conversion types |

### `ConversionType`
#### Description
A type alias that represents the valid conversion types.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ConversionType | "html" \| "pdf" | A union type of the valid conversion types |

### `Input`
#### Description
A type that defines the structure of the input for the document conversion process.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Input | \{ fileName: string; conversionType: ConversionType; bucketName: string \} | An object type with properties for file name, conversion type, and bucket name |

### `Output`
#### Description
A type that defines the structure of the output from the document conversion process.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Output | \{ url: string \} | An object type with a URL property representing the location of the converted document |
