## High-level description
This file defines a custom error class `MetriportError` that extends the built-in `Error` class. It also defines a type `AdditionalInfo` for additional error information. The `MetriportError` class is designed to provide more detailed error information, including an HTTP status code, a cause, and additional context-specific information.

## Symbols

### `AdditionalInfo`
#### Description
A type alias for a record of string keys with values that can be string, number, undefined, or null. It's used to provide additional context-specific information for errors.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| key | string | The name of the additional information field |
| value | string \| number \| undefined \| null | The value of the additional information field |

### `MetriportError`
#### Description
A custom error class that extends the built-in `Error` class. It includes an HTTP status code, a cause, and additional information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The error message |
| cause | unknown | (optional) The underlying cause of the error |
| additionalInfo | AdditionalInfo | (optional) Additional context-specific information |

#### Internal Logic
- Sets the default HTTP status code to 500 (Internal Server Error)
- Stores the provided cause and additional information
- Calls the parent `Error` constructor with the provided message
