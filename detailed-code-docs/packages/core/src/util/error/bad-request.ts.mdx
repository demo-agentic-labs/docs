## High-level description
This file defines a custom error class `BadRequestError` that extends `MetriportError`. It's used to represent errors related to bad requests in the Metriport application. The class is marked as deprecated, suggesting that users should use `@metriport/shared` instead.

## Code Structure
The `BadRequestError` class extends `MetriportError` and sets a specific HTTP status code for bad requests. It also allows for additional information to be passed along with the error message.

## Symbols

### BadRequestError
#### Description
A custom error class for representing bad request errors in the Metriport application. It extends `MetriportError` and sets the HTTP status code to 400 (Bad Request).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The error message (optional, has a default value) |
| cause | unknown | The cause of the error (optional) |
| additionalInfo | Record&lt;string, string \| number \| undefined \| null&gt; | Additional information about the error (optional) |

#### Internal Logic
1. Calls the `super` constructor with the provided message, cause, and additionalInfo.
2. Sets the `status` property to the numeric value of `httpStatus.BAD_REQUEST` (400).
3. Sets the `name` property to the constructor's name ("BadRequestError").
