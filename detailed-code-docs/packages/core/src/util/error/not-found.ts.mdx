## High-level description
This code defines a custom error class `NotFoundError` that extends the `MetriportError` class. It is used to represent errors when a requested resource cannot be found, with a default message and an HTTP status code of 404 (Not Found).

## Code Structure
The `NotFoundError` class extends `MetriportError` and sets specific properties for the error status and name.

## Symbols

### NotFoundError
#### Description
A custom error class for representing "not found" errors in the application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | Optional. The error message. Defaults to "Could not find the requested resource" |
| cause | unknown | Optional. The underlying cause of the error |
| additionalInfo | AdditionalInfo | Optional. Additional information about the error |

#### Internal Logic
- Sets the `status` property to the HTTP status code for "Not Found" (404)
- Sets the `name` property to the constructor name ("NotFoundError")
