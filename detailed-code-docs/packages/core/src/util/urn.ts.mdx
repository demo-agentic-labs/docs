## High-level description
This code provides utility functions for handling Uniform Resource Names (URNs) in the context of UUIDs and OIDs. It includes functions to wrap IDs in URN format, strip URN prefixes, and a regular expression for matching URN prefixes.

## Symbols

### `urnRegex`
#### Description
A regular expression that matches URN prefixes for UUID and OID.

#### Internal Logic
The regex pattern `^urn:(oid|uuid):` matches strings that start with either "urn:oid:" or "urn:uuid:".

___

### `wrapIdInUrnUuid`
#### Description
Wraps a given ID in a URN UUID format.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | The ID to be wrapped |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | string | The ID wrapped in URN UUID format |

#### Internal Logic
Prepends "urn:uuid:" to the input ID.

___

### `wrapIdInUrnOid`
#### Description
Wraps a given ID in a URN OID format.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | The ID to be wrapped |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | string | The ID wrapped in URN OID format |

#### Internal Logic
Prepends "urn:oid:" to the input ID.

___

### `stripUrnPrefix`
#### Description
Removes the URN prefix from a given string or number, if present.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| urn | string \| number \| undefined | The URN to strip the prefix from |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | string | The input with the URN prefix removed, or an empty string if input is undefined |

#### Internal Logic
1. If the input is undefined, return an empty string.
2. If the input is a number, convert it to a string.
3. Use the `replace` method with `urnRegex` to remove the URN prefix if present.
