## High-level description
This code provides utility functions for converting between base64-encoded strings and regular strings or buffers. It includes three main functions: `base64ToBuffer`, `base64ToString`, and `stringToBase64`.

## Symbols

### `base64ToBuffer`
#### Description
Converts a base64-encoded string to a Buffer object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | A base64-encoded string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | Buffer | A Buffer object containing the decoded data |

#### Internal Logic
Uses the `Buffer.from()` method with the 'base64' encoding to convert the input string to a Buffer.

### `base64ToString`
#### Description
Converts a base64-encoded string to a regular string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | A base64-encoded string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | string | A decoded string |

#### Internal Logic
Calls `base64ToBuffer` to convert the input to a Buffer, then uses the `toString()` method to convert the Buffer to a string.

### `stringToBase64`
#### Description
Converts a regular string to a base64-encoded string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | A regular string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | string | A base64-encoded string |

#### Internal Logic
Uses `Buffer.from()` to create a Buffer from the input string, then calls `toString('base64')` to convert it to a base64-encoded string.
