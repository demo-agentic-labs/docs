## High-level description
This file defines a utility function `makeObservation` that creates a FHIR Observation resource with default values and allows for customization through optional parameters. It's primarily used for testing purposes to generate sample Observation data.

## Symbols

### `makeObservation`
#### Description
The `makeObservation` function creates and returns a FHIR Observation resource. It combines default values from `makeBaseDomain` and `makeSubjectReference` functions, sets the `resourceType` to "Observation", and allows for customization through the `params` argument.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;Observation&gt; | Optional. An object containing partial Observation properties to override or add to the default values. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| observation | Observation | A FHIR Observation resource with default values and any provided custom properties. |

#### Internal Logic
1. Spread the result of `makeBaseDomain()` to set base domain properties.
2. Spread the result of `makeSubjectReference()` to set subject reference properties.
3. Set the `resourceType` to "Observation".
4. Spread the `params` object to override or add custom properties.
