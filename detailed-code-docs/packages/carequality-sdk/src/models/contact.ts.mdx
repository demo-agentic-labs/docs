## High-level description
This code defines a Zod schema for a Contact object and exports it along with its corresponding TypeScript type. The schema represents a contact with properties such as purpose, name, telecom information, and address.

## Code Structure
The code imports necessary dependencies and schemas from other files, defines the `contactSchema` using Zod, and exports both the schema and the inferred TypeScript type.

#### Internal Logic
The schema defines an array of objects with the following structure:
1. `purpose`: An optional object value
2. `name`: A name with use information (imported from shared)
3. `telecom`: An array of objects containing system, value, and use information
4. `address`: An address schema (imported from address)
