## High-level description
This JSON file contains AWS CDK context information for various AWS resources across multiple regions. It includes details about availability zones, hosted zones, and VPC configurations for the Metriport.com domain and associated AWS account.

## Symbols

### Availability Zones
#### Description
Lists the available availability zones for specific AWS regions.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| account | string | AWS account ID |
| region | string | AWS region |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| availability zones | array of strings | List of availability zone names |

### Hosted Zones
#### Description
Provides information about the hosted zone for the metriport.com domain in different regions.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| account | string | AWS account ID |
| domainName | string | Domain name |
| region | string | AWS region |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Id | string | Hosted zone ID |
| Name | string | Domain name with trailing dot |

### VPC Configuration
#### Description
Detailed VPC configuration for a specific VPC in the us-east-2 region.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| account | string | AWS account ID |
| filter.vpc-id | string | VPC ID |
| region | string | AWS region |
| returnAsymmetricSubnets | boolean | Flag to return asymmetric subnets |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| vpcId | string | VPC ID |
| vpcCidrBlock | string | VPC CIDR block |
| ownerAccountId | string | AWS account ID |
| availabilityZones | array | Empty in this case |
| subnetGroups | array of objects | Subnet group configurations |

#### Internal Logic
The VPC configuration includes two subnet groups: Private and Public. Each subnet group contains three subnets, one for each availability zone in the us-east-2 region. Each subnet is defined with its ID, CIDR block, availability zone, and associated route table ID.
