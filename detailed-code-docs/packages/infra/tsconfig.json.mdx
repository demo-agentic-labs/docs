## High-level description
This file is a TypeScript configuration file (tsconfig.json) for the "infra" package. It extends the Node.js 18 configuration and includes additional strict compiler options to enhance code quality and type safety.

## Code Structure
The configuration is organized into three main sections: the base configuration extension, compiler options, and file inclusion/exclusion settings.

## Symbols

### `extends`
#### Description
Specifies the base configuration file to extend from.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| @tsconfig/node18/tsconfig.json | string | Path to the Node.js 18 TypeScript configuration file |

### `compilerOptions`
#### Description
Defines the compiler options for the TypeScript project.

#### Internal Logic
1. Extends the Node.js 18 configuration
2. Adds strict compiler options from @tsconfig/strictest (commented for future migration)
3. Configures source map generation, root directory, output directory, and symbol link preservation

### `include`
#### Description
Specifies which folders to include in the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| bin | string | Folder to include in compilation |
| lib | string | Folder to include in compilation |
| config | string | Folder to include in compilation |

### `exclude`
#### Description
Specifies which folders to exclude from the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cdk.out | string | Folder to exclude from compilation |
| dist | string | Folder to exclude from compilation |
| node_modules | string | Folder to exclude from compilation |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| allowUnusedLabels | boolean | false | Disallows unused labels |
| allowUnreachableCode | boolean | false | Disallows unreachable code |
| noFallthroughCasesInSwitch | boolean | true | Reports errors for fallthrough cases in switch statements |
| noImplicitOverride | boolean | true | Ensures overriding members are marked with an override modifier |
| noUnusedLocals | boolean | true | Reports errors on unused local variables |
| checkJs | boolean | true | Enables type checking on JavaScript files |
| noImplicitReturns | boolean | true | Ensures all code paths in a function return a value |
| noUncheckedIndexedAccess | boolean | true | Adds undefined to any un-declared field in an index signature |
| sourceMap | boolean | true | Generates corresponding .map files |
| rootDir | string | "./" | Specifies the root folder within source files |
| outDir | string | "./dist" | Specifies the output folder for all emitted files |
| preserveSymlinks | boolean | true | Preserves symlinks when resolving modules |
