## High-level description
This TypeScript configuration file (`tsconfig.ref.json`) extends the base configuration and sets up specific compiler options for the CommonWell certification runner package. It's designed to be used as a reference in a larger project structure, enabling TypeScript's project references feature.

## Code Structure
The configuration file extends the base `tsconfig.json` and adds specific settings for the CommonWell certification runner package.

## Symbols

### Root Object
#### Description
The root object of the TypeScript configuration file contains three main properties: `extends`, `include`, and `compilerOptions`.

#### Internal Logic
1. Extends the base configuration from `./tsconfig.json`.
2. Specifies files to include in the compilation.
3. Sets compiler options specific to this package.

### `extends`
#### Description
Specifies the base configuration file to extend.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| extends | string | Path to the base configuration file |

### `include`
#### Description
Specifies an array of glob patterns that match files to be included in compilation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| include | string[] | Array of glob patterns |

### `compilerOptions`
#### Description
Specifies the compiler options for this configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| composite | boolean | Enables TypeScript's project references feature |

## Dependencies
This configuration file depends on the base `tsconfig.json` file in the same directory.

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| composite | boolean | true | Enables TypeScript's project references feature |

## References
This configuration file is referenced in the root `tsconfig.json` file of the project, specifically in the `references` array:

```json
{
  "path": "./packages/commonwell-cert-runner/tsconfig.ref.json"
}
```

This reference allows the main project to include this package as a part of its build process, enabling incremental builds and better project structure management.
