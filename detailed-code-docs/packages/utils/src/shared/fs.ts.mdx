## High-level description
This file contains utility functions for file system operations in TypeScript. It provides functions to check if a path is a directory, check if a file exists, get file names from a directory, read file contents, write file contents, and create directories.

## Code Structure
The code exports several functions that utilize the Node.js `fs` and `path` modules to perform various file system operations. These functions are designed to be used as utility methods in other parts of the application.

### getFileNames
#### Description
Retrieves file names from a specified folder, optionally recursively and filtered by extension.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| folder | string | The folder to search |
| recursive | boolean | Whether to search recursively (default: false) |
| extension | string | Optional file extension to filter by |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| fileNames | string[] | Array of file names |

#### Internal Logic
1. Reads the contents of the specified folder
2. Filters the contents based on the provided criteria (file type and extension)
3. If recursive, it repeats the process for subdirectories
4. Returns an array of file names
