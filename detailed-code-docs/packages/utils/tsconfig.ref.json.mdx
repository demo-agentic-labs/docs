## High-level description
This TypeScript configuration file (`tsconfig.ref.json`) extends the base configuration and sets up specific compiler options for the `utils` package. It's designed to be used as a reference in a project with multiple packages, enabling TypeScript's project references feature.

## Code Structure
The configuration file extends the base `tsconfig.json` and adds specific settings for the `utils` package.

## Symbols

### Root Object
#### Description
The root object of the TypeScript configuration file for the `utils` package.

#### Internal Logic
1. Extends the base `tsconfig.json` configuration.
2. Specifies files to include in the compilation.
3. Sets compiler options specific to this package.

### `extends`
#### Description
Specifies the base configuration file to extend.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| extends | string | Path to the base configuration file |

### `include`
#### Description
Specifies which files should be included in the compilation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| include | array of strings | Glob patterns for files to include |

### `compilerOptions`
#### Description
Specifies the compiler options for this package.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| composite | boolean | Enables TypeScript's project references feature |

## Dependencies
