## High-level description
This file is a TypeScript configuration file (tsconfig.json) for the "utils" package. It defines the compiler options and file inclusion/exclusion rules for TypeScript compilation in this project.

## Code Structure
The configuration is structured as a JSON object with four main properties: "extends", "compilerOptions", "include", and "exclude".

## Symbols

### Root Configuration Object
#### Description
The root object in this JSON file contains the entire TypeScript configuration for the "utils" package.

#### Internal Logic
1. Extends a base configuration
2. Specifies compiler options
3. Defines which files to include in compilation
4. Specifies which directories to exclude from compilation

### `extends`
#### Description
This property specifies a base configuration file to inherit from, using community best practices.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| @tsconfig/node18/tsconfig.json | string | Path to the base configuration file |

### `compilerOptions`
#### Description
This object specifies the compiler options for TypeScript compilation.

#### Internal Logic
1. Sets the output directory for compiled files
2. Enables resolving of JSON modules

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| outDir | string | Specifies the output directory for compiled files |
| resolveJsonModule | boolean | Enables importing JSON files as modules |

### `include`
#### Description
This property specifies which files should be included in the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ./src | string | Includes all files in the src directory |

### `exclude`
#### Description
This property specifies which directories should be excluded from the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| dist | string | Excludes the dist directory |
| node_modules | string | Excludes the node_modules directory |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @tsconfig/node18 | Provides a base TypeScript configuration for Node.js 18 |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| outDir | string | "./dist" | Specifies the output directory for compiled files |
| resolveJsonModule | boolean | true | Enables importing JSON files as modules |

This TypeScript configuration file sets up a standard environment for a Node.js 18 project, with output directed to a "dist" folder, JSON module resolution enabled, and compilation focused on the "src" directory while excluding "dist" and "node_modules". It leverages community best practices by extending a base configuration for Node.js 18.
