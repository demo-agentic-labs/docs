## High-level description
This `package.json` file defines the configuration for the `@metriport/commonwell-sdk` package. It's an SDK designed to simplify CommonWell API integration, developed by Metriport Inc. The package is part of a larger monorepo project and includes TypeScript compilation, linting, and other development scripts.
