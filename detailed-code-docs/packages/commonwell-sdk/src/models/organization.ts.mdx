Here's a high-level description and documentation of the `organization.ts` file:

## High-level description
This file contains functions for managing organizations in the CommonWell Health Alliance system. It provides functionality to create, update, and retrieve organization information, as well as handle the integration with the CareQuality (CQ) bridge.

## Code Structure
The main functions in this file interact with the CommonWell API to perform operations on organizations. These functions use the `makeCommonWellAPI` and `makeCommonWellManagementAPI` to create API instances for interacting with CommonWell.

## Symbols

### `organizationToCommonwell`
#### Description
Converts an internal organization representation to the CommonWell SDK organization format.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| org | CWOrganization | The organization to convert |
| isObo | boolean | Flag indicating if this is an "on behalf of" organization |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cwOrg | CWSdkOrganizationWithOrgId | The converted CommonWell organization |

### `get`
#### Description
Retrieves an organization from CommonWell by its OID.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| orgOid | string | The OID of the organization to retrieve |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| organization | CWSdkOrganization \| undefined | The retrieved organization or undefined if not found |

### `create`
#### Description
Creates a new organization in CommonWell.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cxId | string | The customer ID |
| org | CWOrganization | The organization to create |
| isObo | boolean | Flag indicating if this is an "on behalf of" organization |

### `update`
#### Description
Updates an existing organization in CommonWell.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cxId | string | The customer ID |
| org | CWOrganization | The organization to update |
| isObo | boolean | Flag indicating if this is an "on behalf of" organization |

### `initCQOrgIncludeList`
#### Description
Initializes the CareQuality organization include list for a given organization.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| orgOid | string | The OID of the organization |

### `parseCWEntry`
#### Description
Parses a CommonWell SDK organization into the internal CWOrganization format.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| org | CWSdkOrganization | The CommonWell SDK organization to parse |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| parsedOrg | CWOrganization | The parsed organization |

### `getParsedOrgOrFail`
#### Description
Retrieves and parses an organization from CommonWell, throwing an error if not found.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| oid | string | The OID of the organization to retrieve |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| parsedOrg | CWOrganization | The retrieved and parsed organization |

## Error Handling
The functions in this file use try-catch blocks to handle errors. When an error occurs, it is logged and captured using a notification system. In some cases, specific error handling is implemented, such as creating an organization if it's not found during an update operation.

## Dependencies
This file depends on various external modules and internal configurations, including the CommonWell SDK, CareQuality bridge, and environment-specific configurations.
