## High-level description
This file defines the data model for demographics information in the CommonWell SDK. It includes schemas for gender, demographics, and related types using the Zod library for runtime type checking and validation.

## Code Structure
The file defines several interconnected schemas:
- `GenderCodes`: An enum for gender codes
- `genderSchema`: A schema for gender information
- `demographicsSchema`: A schema for demographic details
- Various imported schemas for related data types (e.g., identifiers, human names, contacts, addresses)

## Symbols

### `GenderCodes`
#### Description
An enum representing gender codes used in the CommonWell system.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| F | string | Female |
| M | string | Male |
| UN | string | Undifferentiated |

### `genderCodesSchema`
#### Description
A Zod schema for validating gender codes based on the `GenderCodes` enum.

### `genderSchema`
#### Description
A Zod schema for validating gender information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| code | GenderCodes | The gender code |
| display | string (optional) | Display text for the gender |
| system | string (optional) | The system used for the gender code |

### `demographicsSchema`
#### Description
A Zod schema for validating demographic details of a person.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| identifier | array of identifierSchema (optional) | Identifiers for the person |
| name | array of humanNameSchema (min 1) | Names of the person |
| telecom | array of contactSchema (optional) | Contact information |
| gender | genderSchema | Gender information |
| birthDate | isoDateTimeSchema or isoDateSchema | Date of birth |
| address | array of addressSchema | Addresses |
| picture | any (optional) | Picture (not supported) |

## Types
The file exports two types:
- `Gender`: Inferred from `genderSchema`
- `Demographics`: Inferred from `demographicsSchema`

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| zod | Schema validation and type inference |
| ./iso-datetime | Schema for ISO date-time format |
| ./identifier | Schema for identifiers |
| ./human-name | Schema for human names |
| ./contact | Schema for contact information |
| ./address | Schema for addresses |
| ./iso-date | Schema for ISO date format |

This file provides a structured way to represent and validate demographic information in the CommonWell SDK, ensuring data consistency and type safety throughout the application.
