## High-level description
This file contains test utility functions for creating mock Document and DocumentContent objects, as well as related contained resources (Organization, Practitioner, Patient) for testing purposes. It uses the Faker library to generate realistic mock data.

## Code Structure
The main functions `makeDocument` and `makeDocumentContent` create mock Document and DocumentContent objects respectively. These are supported by helper functions like `makeContained`, `makeContainedOrg`, `makeContainedPractitioner`, and `makeContainedPatient` which create mock contained resources.

## Symbols

### `makeDocument`
#### Description
Creates a mock Document object with an optional id and content.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;Pick&lt;Document, "id"&gt;&gt; & \{ content?: MakeDocumentContent \} | Optional parameters to customize the document |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| document | Document | A mock Document object |

#### Internal Logic
1. Generates a random UUID for the document id if not provided
2. Calls `makeDocumentContent` to create the content

### `makeDocumentContent`
#### Description
Creates a mock DocumentContent object with optional customizations.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | MakeDocumentContent | Optional parameters to customize the document content |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| content | DocumentContent | A mock DocumentContent object |

#### Internal Logic
1. Generates random IDs for contained resources
2. Creates mock contained resources (Organization and Practitioner)
3. Generates random dates for the context period
4. Populates all required fields with either provided values or generated mock data

### `makeContained`
#### Description
Creates a mock Contained resource (Organization, Practitioner, or Patient) randomly.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;Contained&gt; | Optional parameters to customize the contained resource |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| contained | Contained | A mock Contained resource |

### `makeContainedOrg`, `makeContainedPractitioner`, `makeContainedPatient`
#### Description
Create specific types of mock Contained resources.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;Contained&gt; | Optional parameters to customize the contained resource |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| contained | Contained | A mock Contained resource of the specific type |
