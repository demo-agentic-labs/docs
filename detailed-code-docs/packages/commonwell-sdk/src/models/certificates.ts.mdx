## High-level description
This file defines TypeScript models and schemas for certificates in the CommonWell SDK. It uses Zod for schema validation and type inference, providing a structured way to handle certificate data, including their purposes, parameters, and responses.

## Code Structure
The code defines several interconnected schemas and types:
- `CertificatePurpose`: An enum for certificate purposes.
- `certificateSchema`: Defines the structure of a certificate.
- `certificatesParamSchema`: Defines the structure for certificate parameters.
- `certificateRespSchema`: Defines the structure for certificate responses.

These schemas are used to create TypeScript types through Zod's type inference.
