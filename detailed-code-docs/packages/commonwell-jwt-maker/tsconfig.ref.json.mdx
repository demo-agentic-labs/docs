## High-level description
This TypeScript configuration file (`tsconfig.ref.json`) extends the base configuration and specifies settings for the CommonWell JWT Maker package. It sets up the compilation process for TypeScript files in the `src` directory and enables project references for use in a larger monorepo structure.

## Code Structure
The configuration file extends the base `tsconfig.json` and adds specific settings for the CommonWell JWT Maker package.

## Symbols

### Root Configuration Object
#### Description
The root configuration object defines the TypeScript compiler options and file inclusion settings for the CommonWell JWT Maker package.

1. Extends the base `tsconfig.json` configuration.

3. Sets compiler options specific to this package.
