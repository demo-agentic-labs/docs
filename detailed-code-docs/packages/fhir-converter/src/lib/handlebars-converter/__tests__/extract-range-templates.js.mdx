## High-level description
This file contains test data and input/output pairs for various scenarios related to extracting reference ranges and building presented forms in a FHIR converter. It defines several objects representing different data structures and their expected outputs, which are likely used in unit tests for the FHIR converter's handlebars helpers.

## Symbols

#### Description
Represents an input object for a reference range with an interval of physical quantities (IVL_PQ).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| text | object | Contains the textual representation of the range |
| value | object | Contains the structured representation of the range |
| interpretationCode | object | Contains the interpretation code for the range |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| typeIvlPqOutput | object | Expected output object for the `typeIvlPq` input |

## Notes

- The various input objects represent different scenarios that the `extractReferenceRange` function might encounter, such as different data types, null flavors, and edge cases.
