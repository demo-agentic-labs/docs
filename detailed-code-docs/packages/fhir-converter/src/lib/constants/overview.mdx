## High-level description
This file defines a set of constants used in the FHIR Converter project. It specifies file paths for template locations, keys for global scope operations, and namespace constants for the conversion process.

## Symbols

### `BASE_TEMPLATE_FILES_LOCATION`
#### Description
Defines the base path for template files used in the FHIR Converter.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| BASE_TEMPLATE_FILES_LOCATION | string | Absolute path to the base template files directory |

### `TEMPLATE_FILES_LOCATION`
#### Description
Specifies the path for service-specific template files.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| TEMPLATE_FILES_LOCATION | string | Absolute path to the service-specific template files directory |

### `MOVE_TO_GLOBAL_KEY_NAME`
#### Description
Defines a key name used to indicate that a resource should be moved to the global scope.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| MOVE_TO_GLOBAL_KEY_NAME | string | Key name for moving resources to global scope |

### `CDA_TEMPLATE_LOCATION`
#### Description
Specifies the path for CDA (Clinical Document Architecture) specific template files.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CDA_TEMPLATE_LOCATION | string | Absolute path to the CDA-specific template files directory |

### `CLS_NAMESPACE`
#### Description
Defines the namespace used for the conversion request.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CLS_NAMESPACE | string | Namespace identifier for the conversion request |

### `CLS_KEY_HANDLEBAR_INSTANCE`
#### Description
Specifies the key used to store the Handlebars instance in the conversion context.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CLS_KEY_HANDLEBAR_INSTANCE | string | Key for storing the Handlebars instance |

### `CLS_KEY_TEMPLATE_LOCATION`
#### Description
Defines the key used to store the template location in the conversion context.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CLS_KEY_TEMPLATE_LOCATION | string | Key for storing the template location |

## Dependencies
