## High-level description
This file defines test inputs for an encounter deduplication process in a FHIR (Fast Healthcare Interoperability Resources) context. It creates sample encounter resources with specific attributes to test the merging of duplicate encounters based on shared identifiers.

## Symbols

### `bundleTemplate`
#### Description
Defines a basic structure for a FHIR Bundle resource of type "collection".

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| bundleTemplate | Object | A template for a FHIR Bundle resource |

### `sameIdentifier`
#### Description
A UUID generated to represent a shared identifier between encounters.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| sameIdentifier | String | A UUID used as a shared identifier |

### `idToKeep`
#### Description
A UUID generated to represent the ID of the encounter that should be kept after deduplication.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| idToKeep | String | A UUID used as the ID for the encounter to keep |

### `encompassingEncounter`
#### Description
Defines a sample FHIR Encounter resource representing an encompassing encounter.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| encompassingEncounter | Object | A FHIR Encounter resource with a shared identifier and service provider |

### `internalEncounter`
#### Description
Defines a sample FHIR Encounter resource representing an internal encounter.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| internalEncounter | Object | A FHIR Encounter resource with a shared identifier and subject reference |

### `mergedEncounter`
#### Description
Defines the expected result of merging `encompassingEncounter` and `internalEncounter`.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mergedEncounter | Object | A FHIR Encounter resource combining properties from both input encounters |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @faker-js/faker | Used to generate random UUIDs for identifiers and IDs |

This file is crucial for testing the encounter deduplication logic in the FHIR converter. It provides carefully crafted input data and the expected output after deduplication, allowing for comprehensive testing of the deduplication process.
