## High-level description
This Handlebars template is designed to convert and format quantity data into a standardized JSON structure. It handles various input formats, including feet and inches, and extracts numerical values and units from the input, providing a consistent output format for quantity measurements.

## Symbols

### decodeHtmlEntities
#### Description
This helper function decodes HTML entities in the input string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| quantity.value | string | The input quantity value, potentially containing HTML entities |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decodedValue | string | The decoded quantity value |

### convertFeetAndInchesToCm
#### Description
This helper function converts feet and inches measurements to centimeters.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decodedValue | string | The decoded quantity value |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| feetAndInches | object | An object containing the converted value in cm and a validity flag |

### extractNumberAndUnit
#### Description
This helper function extracts the numerical value and unit from the input string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decodedValue | string | The decoded quantity value |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| numberAndUnit | object | An object containing the extracted value, unit, and a validity flag |

### extractDecimal
#### Description
This helper function extracts a decimal number from the input string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decodedValue | string | The decoded quantity value |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decimal | number | The extracted decimal value |

## Internal Logic
1. The template first decodes any HTML entities in the input quantity value.
2. It then attempts to convert feet and inches to centimeters.
3. If the feet and inches conversion is valid, it outputs the result in centimeters.
4. If not, it tries to extract a number and unit from the input.
5. If successful, it uses the extracted number and either the provided unit or the extracted unit.
6. If unsuccessful, it attempts to extract a decimal number and uses the provided unit.
7. In all cases, the output is formatted as a JSON object with "value", "unit", and "system" properties.

## Dependencies
This template relies on several custom Handlebars helper functions:
| Dependency | Purpose |
|:-----------|:--------|
| decodeHtmlEntities | Decodes HTML entities in the input string |
| convertFeetAndInchesToCm | Converts feet and inches measurements to centimeters |
| extractNumberAndUnit | Extracts numerical value and unit from the input string |
| extractDecimal | Extracts a decimal number from the input string |

These helper functions are not defined in this file and must be provided elsewhere in the application.
