## High-level description
This Handlebars template defines the structure for an ExplanationOfBenefit resource in FHIR format, specifically focusing on the procedure reference within the resource. It creates a JSON object that represents a single procedure reference within the ExplanationOfBenefit resource.

## Symbols

### ExplanationOfBenefit Resource Template
#### Description
This template creates a JSON structure for an ExplanationOfBenefit resource with a focus on the procedure reference. It includes the resource type, an ID, and a procedure array containing a single procedure reference.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID   | string | The unique identifier for the ExplanationOfBenefit resource |
| REF  | string | The reference to the procedure |

#### Outputs
The output is a JSON object representing an ExplanationOfBenefit resource with a procedure reference.

#### Internal Logic
1. Sets the resourceType to "ExplanationOfBenefit"
2. Assigns the provided ID to the resource
3. Creates a procedure array with a single object
4. Within the procedure object, creates a procedureReference object with a reference field

## Dependencies
This template relies on the Handlebars templating engine for rendering the JSON structure with dynamic values.

## Configuration
The template uses two configuration options:
| Option | Type | Description |
|:-------|:-----|:------------|
| ID     | string | The identifier for the ExplanationOfBenefit resource |
| REF    | string | The reference value for the procedure |
