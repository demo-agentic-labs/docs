## High-level description
This Handlebars template defines a partial structure for an ExplanationOfBenefit FHIR resource, specifically focusing on the procedure's Unique Device Identifier (UDI) reference. It's designed to be used in the context of converting Clinical Document Architecture (CDA) data to FHIR format.

## Symbols

### ExplanationOfBenefit Resource Template
#### Description
This template creates a partial JSON structure for an ExplanationOfBenefit resource, emphasizing the procedure's UDI reference.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID   | String | The identifier for the ExplanationOfBenefit resource |
| REF  | String | The reference for the procedure's UDI |

#### Outputs
A JSON object representing a partial ExplanationOfBenefit resource with a procedure containing a UDI reference.

#### Internal Logic
1. Creates an ExplanationOfBenefit resource structure.
2. Sets the resource type to "ExplanationOfBenefit".
3. Assigns the provided ID to the resource.
4. Creates a procedure array with a single element.
5. Within the procedure, creates a UDI array with a single element containing the provided reference.
