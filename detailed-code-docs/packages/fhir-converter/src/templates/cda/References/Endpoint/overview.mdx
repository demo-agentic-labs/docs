## High-level description
This Handlebars template file defines the structure for creating an Endpoint resource in FHIR format, specifically focusing on the `managingOrganization` field. It's designed to be used in a FHIR converter, likely for transforming CDA (Clinical Document Architecture) data to FHIR format.

## Symbols

### Unnamed Handlebars Template
#### Description
This template generates a JSON structure for an Endpoint resource in FHIR format. It specifically populates the `managingOrganization` field of the Endpoint resource.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | string | The identifier for the Endpoint resource |
| REF | string | The reference to the managing organization |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON object | object | A partial FHIR Endpoint resource with managingOrganization field |

#### Internal Logic
1. Creates a JSON object with a `resource` property.
2. Sets the `resourceType` to "Endpoint".
3. Assigns the provided `ID` to the `id` field.
4. Creates a `managingOrganization` object with a `reference` field, populated by the `REF` input.
