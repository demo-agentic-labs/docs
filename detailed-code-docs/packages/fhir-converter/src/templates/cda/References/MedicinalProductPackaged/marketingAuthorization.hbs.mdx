## High-level description
This Handlebars template defines a FHIR (Fast Healthcare Interoperability Resources) MedicinalProductPackaged resource with a marketing authorization reference. It's designed to be used in a FHIR converter, likely for transforming CDA (Clinical Document Architecture) data to FHIR format.

## Symbols

### MedicinalProductPackaged Template
#### Description
This template creates a JSON structure for a MedicinalProductPackaged resource, which includes an ID and a marketing authorization reference.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | string | The identifier for the MedicinalProductPackaged resource |
| REF | string | The reference to the marketing authorization |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON object | object | A FHIR-compliant MedicinalProductPackaged resource |

#### Internal Logic
1. The template creates a JSON object with a "resource" property.
2. Inside the "resource" object, it sets:
   - "resourceType" as "MedicinalProductPackaged"
   - "id" using the provided `{{ID}}` value
   - "marketingAuthorization" object with a "reference" property set to the provided `{{REF}}` value
