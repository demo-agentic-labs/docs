## High-level description
This file contains a Handlebars template that defines the structure of a FHIR MedicinalProductPackaged resource. It specifically focuses on the `packageItem.manufacturedItem` element, providing a JSON structure for this resource type.

## Symbols

### MedicinalProductPackaged Template
#### Description
This template generates a JSON structure for a FHIR MedicinalProductPackaged resource, specifically focusing on the `packageItem.manufacturedItem` element.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID   | String | The unique identifier for the MedicinalProductPackaged resource |
| REF  | String | The reference to the manufactured item |

#### Outputs
The output is a JSON object representing a FHIR MedicinalProductPackaged resource with the following structure:

```json
{
    "resource": {
        "resourceType": "MedicinalProductPackaged",
        "id": "{{ID}}",
        "packageItem": [
            {
                "manufacturedItem": [
                    {
                        "reference": "{{REF}}",
                    },
                ],
            },
        ],
    },
}
```

#### Internal Logic
1. The template sets the `resourceType` to "MedicinalProductPackaged".
2. It uses the `{{ID}}` placeholder for the resource's `id`.
3. Within the `packageItem` array, it defines a single item with a `manufacturedItem` array.
4. The `manufacturedItem` array contains a single object with a `reference` field, using the `{{REF}}` placeholder.

## Dependencies
This template relies on the Handlebars templating engine to process the placeholders ({{ID}} and {{REF}}).

There are no specific configuration options defined in this template.

This template does not implement any explicit error handling. It assumes that the `ID` and `REF` values will be provided when the template is rendered.
