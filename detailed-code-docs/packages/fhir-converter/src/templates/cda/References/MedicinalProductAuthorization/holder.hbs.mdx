## High-level description
This Handlebars template defines a FHIR (Fast Healthcare Interoperability Resources) MedicinalProductAuthorization resource with a holder reference. It's designed to generate a JSON structure representing a medicinal product authorization with a specific holder.

## Symbols

### MedicinalProductAuthorization Template
#### Description
This template creates a JSON structure for a MedicinalProductAuthorization FHIR resource. It includes an ID and a holder reference.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | String | The unique identifier for the MedicinalProductAuthorization resource |
| REF | String | The reference to the holder of the medicinal product authorization |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON | Object | A JSON object representing a MedicinalProductAuthorization FHIR resource |

#### Internal Logic
1. The template creates a JSON object with a "resource" property.

   - "id" using the provided `{{ID}}` input
   - "holder" object with a "reference" property set to the provided `{{REF}}` input
