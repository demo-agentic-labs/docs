## High-level description
This Handlebars template defines the structure for a FHIR DetectedIssue resource, specifically focusing on the patient reference. It's designed to create a JSON object representing a detected issue associated with a patient in a healthcare context.

## Symbols

### DetectedIssue Template
#### Description
This template generates a JSON object for a FHIR DetectedIssue resource. It includes the resource type, an ID, and a reference to the associated patient.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | String | The unique identifier for the DetectedIssue resource |
| REF | String | The reference to the associated patient |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON Object | Object | A structured JSON object representing a FHIR DetectedIssue resource |

#### Internal Logic
1. Sets the `resourceType` to "DetectedIssue"
2. Assigns the provided `ID` to the `id` field
3. Creates a `patient` object with a `reference` field containing the provided `REF` value

## Dependencies
This template relies on the Handlebars templating engine to process the dynamic content.

No specific configuration options are defined in this template.
