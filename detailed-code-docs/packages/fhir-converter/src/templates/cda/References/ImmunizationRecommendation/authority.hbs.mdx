## High-level description
This Handlebars template defines the structure for an ImmunizationRecommendation resource in FHIR format. It specifically focuses on the 'authority' field of the ImmunizationRecommendation, providing a reference to the authority responsible for the recommendation.

## Symbols

### ImmunizationRecommendation Template
#### Description
This template creates a JSON structure for an ImmunizationRecommendation resource, including its resourceType, id, and authority reference.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | string | The unique identifier for the ImmunizationRecommendation resource |
| REF | string | The reference to the authority responsible for the recommendation |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON object | object | A structured JSON object representing an ImmunizationRecommendation resource |

#### Internal Logic
1. Sets the resourceType to "ImmunizationRecommendation"
2. Assigns the provided ID to the resource's id field
3. Creates an authority object with a reference field, populated by the provided REF value

## Dependencies
This template relies on the Handlebars templating engine to process the placeholders ({{ID}} and {{REF}}).

## Configuration
The template doesn't include any specific configuration options. It uses a fixed structure for the ImmunizationRecommendation resource.
