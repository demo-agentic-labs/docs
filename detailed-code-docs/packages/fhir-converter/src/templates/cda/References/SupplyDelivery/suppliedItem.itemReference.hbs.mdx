## High-level description
This Handlebars template defines the structure for a FHIR SupplyDelivery resource, specifically focusing on the `suppliedItem.itemReference` element. It creates a JSON object representing a SupplyDelivery resource with a supplied item referenced by its identifier.

## Symbols

### SupplyDelivery Resource Template
#### Description
This template creates a JSON object structure for a FHIR SupplyDelivery resource. It specifically defines the `suppliedItem.itemReference` element within the SupplyDelivery resource.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID   | String | The unique identifier for the SupplyDelivery resource |
| REF  | String | The reference to the supplied item |

#### Outputs
A JSON object representing a FHIR SupplyDelivery resource with the following structure:
```json
{
    "resource": {
        "resourceType": "SupplyDelivery",
        "id": "{{ID}}",
        "suppliedItem": {
            "itemReference": {
                "reference": "{{REF}}"
            }
        }
    }
}
```

#### Internal Logic
1. Sets the `resourceType` to "SupplyDelivery"
2. Assigns the provided `ID` to the `id` field
3. Creates a `suppliedItem` object with an `itemReference` sub-object
4. Sets the `reference` field of `itemReference` to the provided `REF` value

## Dependencies
This template relies on the Handlebars templating engine to process the placeholders ({{ID}} and {{REF}}).

## Configuration
No specific configuration options are defined in this template.

## Error Handling
This template does not implement any explicit error handling. It assumes that the `ID` and `REF` values will be provided when the template is rendered.
