## High-level description
This Handlebars template defines the structure for an InsurancePlan resource in FHIR format, specifically focusing on the "ownedBy" field. It's designed to be used in a FHIR converter, likely for transforming CDA (Clinical Document Architecture) data to FHIR format.

## Symbols

### InsurancePlan Resource Template
#### Description
This template creates a JSON structure for an InsurancePlan resource, including the resourceType, id, and ownedBy fields.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | String | The unique identifier for the InsurancePlan resource |
| REF | String | The reference to the organization that owns the insurance plan |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON Object | Object | A structured JSON object representing an InsurancePlan resource |

#### Internal Logic
1. Sets the resourceType to "InsurancePlan"
2. Assigns the provided ID to the id field
3. Creates an ownedBy object with a reference field containing the provided REF value

## Dependencies
This template relies on the Handlebars templating engine for processing.

## Configuration
The template uses Handlebars syntax for variable interpolation:
| Option | Type | Description |
|:-------|:-----|:------------|
| {{ID}} | String | Placeholder for the InsurancePlan's unique identifier |
| {{REF}} | String | Placeholder for the reference to the owning organization |
