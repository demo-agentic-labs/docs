## High-level description
This Handlebars template defines the structure for an InsurancePlan resource in FHIR format, specifically focusing on the "administeredBy" field. It creates a JSON object representing an InsurancePlan with an ID and a reference to the entity that administers the plan.

## Symbols

### InsurancePlan Template
#### Description
This template generates a JSON object for an InsurancePlan resource in FHIR format. It includes the resource type, an ID, and an "administeredBy" field that references the entity administering the plan.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | String | The unique identifier for the InsurancePlan resource |
| REF | String | The reference to the entity administering the plan |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON Object | Object | A JSON object representing an InsurancePlan resource with "administeredBy" information |

#### Internal Logic
1. Sets the `resourceType` to "InsurancePlan"
2. Assigns the provided `ID` to the `id` field
3. Creates an `administeredBy` object with a `reference` field containing the provided `REF` value

## Dependencies
This template relies on the Handlebars templating engine to process the placeholders ({{ID}} and {{REF}}).
