## High-level description
This Handlebars template defines a partial structure for an AuditEvent FHIR resource, specifically focusing on the `source.observer` element. It creates a JSON object representing an AuditEvent with a source observer reference.

## Symbols

### AuditEvent Resource Template
#### Description
This template creates a partial JSON structure for an AuditEvent FHIR resource. It specifically defines the `source.observer` element within the AuditEvent structure.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID   | string | The identifier for the AuditEvent resource |
| REF  | string | The reference value for the observer |

#### Outputs
The template outputs a JSON object representing a partial AuditEvent FHIR resource with the following structure:
- `resource`: The main container for the FHIR resource
  - `resourceType`: Set to "AuditEvent"
  - `id`: The identifier of the AuditEvent
  - `source`: Contains information about the source of the audit event
    - `observer`: Specifies the observer of the audit event
      - `reference`: A reference to the observer

#### Internal Logic
1. The template creates a nested JSON structure for the AuditEvent resource.
2. It sets the `resourceType` to "AuditEvent".
3. It uses the `{{ID}}` placeholder to set the `id` of the AuditEvent.
4. Within the `source` object, it creates an `observer` object.
5. The `observer` object contains a `reference` field, which is populated with the `{{REF}}` placeholder.
