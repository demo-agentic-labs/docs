## High-level description
This Handlebars template defines the structure for an OrganizationAffiliation resource in FHIR format. It specifically focuses on the participatingOrganization field, which is a reference to another organization involved in the affiliation.

## Symbols

### OrganizationAffiliation Template
#### Description
This template creates a JSON structure for an OrganizationAffiliation resource, including the resourceType, id, and participatingOrganization fields.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | string | The unique identifier for the OrganizationAffiliation resource |
| REF | string | The reference to the participating organization |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON | object | A JSON object representing the OrganizationAffiliation resource |

#### Internal Logic
1. Sets the resourceType to "OrganizationAffiliation"
2. Assigns the provided ID to the id field
3. Creates a participatingOrganization object with a reference field containing the provided REF value

## Dependencies
This template relies on the Handlebars templating engine for rendering.

## Configuration
No specific configuration options are defined in this template.

## Error Handling
No explicit error handling is implemented in this template. It assumes that the required input values (ID and REF) are provided correctly.
