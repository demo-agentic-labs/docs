## High-level description
This Handlebars template defines a FHIR MedicationRequest resource with an eventHistory property. It's designed to be used in a FHIR converter, specifically for converting CDA (Clinical Document Architecture) data to FHIR format.

## Symbols

### MedicationRequest Template
#### Description
This template creates a FHIR MedicationRequest resource with an eventHistory array containing a single reference.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | String | The identifier for the MedicationRequest resource |
| REF | String | The reference value for the eventHistory |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON object | Object | A FHIR-compliant MedicationRequest resource |

#### Internal Logic
1. Creates a MedicationRequest resource with the provided ID.
2. Adds an eventHistory array with a single object containing a reference.
3. The reference value is set using the provided REF input.
