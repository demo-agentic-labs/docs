## High-level description
This Handlebars template defines a FHIR MedicationRequest resource with a medicationReference. It's designed to be used in a FHIR converter, specifically for converting CDA (Clinical Document Architecture) data to FHIR format.

## Symbols

### MedicationRequest Template
#### Description
This template creates a FHIR MedicationRequest resource with a medicationReference. It's a partial template, likely to be used as part of a larger FHIR document or bundle.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | String | The unique identifier for the MedicationRequest resource |
| REF | String | The reference to the medication |

#### Outputs
A JSON object representing a FHIR MedicationRequest resource with a medicationReference.

#### Internal Logic
The template creates a nested JSON structure with the following key elements:
1. A "resource" object containing:
   - "resourceType" set to "MedicationRequest"
   - "id" set to the provided ID
   - "medicationReference" object with a "reference" field set to the provided REF

## Dependencies
This template is part of the FHIR Converter project and is designed to work within the Handlebars templating system.

## Configuration
The template uses two configuration variables:
| Option | Type | Description |
|:-------|:-----|:------------|
| ID | String | The identifier for the MedicationRequest resource |
| REF | String | The reference to the medication |
