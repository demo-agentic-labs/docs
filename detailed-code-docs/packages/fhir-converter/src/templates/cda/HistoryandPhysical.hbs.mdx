## High-level description
This Handlebars template file defines the structure of a FHIR Bundle resource for a History and Physical document. It includes various sections such as allergies, family history, medications, results, social history, vital signs, immunizations, problems, and procedures, along with a document reference.

## Code Structure
The template uses Handlebars partials to include different sections of the document. It defines a FHIR Bundle resource with a "batch" type and includes both required and optional sections.

## Symbols

### FHIR Bundle
#### Description
The main structure of the template, which creates a FHIR Bundle resource of type "batch".

#### Internal Logic
1. Includes the Header partial
2. Includes required sections (Allergies, Family History, Medication, Results, Social History, Vital Signs)
3. Includes optional sections (Immunization, Problems, Procedures)
4. Creates a DocumentReference resource

#### Description
Creates a DocumentReference resource for the History and Physical document.
