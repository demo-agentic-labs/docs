## High-level description
This Handlebars template file defines the structure for creating a FHIR Medication resource. It generates a JSON object representing a Medication resource with its associated request information for a PUT operation.

## Symbols

### Medication Resource Template
#### Description
This template creates a JSON object representing a FHIR Medication resource and its associated request information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ID | string | Unique identifier for the Medication resource |
| medication.code | object | Code information for the medication |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSON object | object | A JSON object representing a FHIR Medication resource with request information |

#### Internal Logic
1. Creates a `fullUrl` using the provided `ID`.
2. Defines the `resource` object with:
   - `resourceType` set to "Medication"
   - `id` set to the provided `ID`
   - `code` populated using a partial template `DataType/CodeableConcept.hbs` with the `medication.code` input
3. Defines the `request` object with:
   - `method` set to "PUT"
   - `url` set to "Medication/{ID}"
