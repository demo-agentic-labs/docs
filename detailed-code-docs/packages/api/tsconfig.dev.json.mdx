### tsconfig.dev.json
#### Description
This JSON configuration file extends the base TypeScript configuration (`tsconfig.json`) and modifies compiler options for development environments. It enables source maps and disables checks for unused locals and parameters.

#### Internal Logic

2. It overrides specific compiler options:

   - Disables checks for unused local variables
   - Disables checks for unused parameters
