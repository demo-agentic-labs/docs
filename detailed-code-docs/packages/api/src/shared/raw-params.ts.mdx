## High-level description
This code defines types and functions for handling raw parameters from HTTP requests in an Express.js application. It provides a structured way to access query parameters and headers, which is useful for processing incoming requests in the API.

## Symbols

### `RawParams`
#### Description
A type definition that represents the structure of raw parameters from an HTTP request.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| query | Record&lt;string, string \| string[] \| undefined&gt; | Query parameters from the request |
| headers | Record&lt;string, string \| string[] \| undefined&gt; | Headers from the request |

### `getRawParams`
#### Description
A function that extracts query parameters and headers from an Express Request object and returns them in a structured format.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| req | Request | An Express Request object |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| RawParams | \{ query: Record&lt;string, string \| string[] \| undefined&gt;, headers: Record&lt;string, string \| string[] \| undefined&gt; \} | An object containing the query parameters and headers |

#### Internal Logic
1. The function creates a new object with two properties: `query` and `headers`.
2. It uses the spread operator to copy the `query` and `headers` properties from the input `req` object.
3. The copied properties are type-cast to match the `RawParams` type definition.
