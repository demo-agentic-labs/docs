## High-level description
This file contains utility functions for handling dates and times in the application. It uses the dayjs library for date manipulation and defines constants and functions for formatting, parsing, and validating ISO dates.

## Symbols

### `ISO_DATE`
#### Description
A constant string representing the format for ISO dates (YYYY-MM-DD).

### `formatStartDate`
#### Description
Formats a given date string to an ISO date-time string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | string | The input date string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formattedDate | string | The formatted ISO date-time string |

### `formatEndDate`
#### Description
Formats a given date string to an ISO date-time string, adding 24 hours to the input date.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | string | The input date string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formattedDate | string | The formatted ISO date-time string with 24 hours added |

### `getStartAndEndDateTime`
#### Description
Returns an object with start and end date-times for a given date.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | string | The input date string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | object | An object containing start_date and end_date as ISO date-time strings |

### `getStartAndEndDate`
#### Description
Returns an object with start and end dates for a given date.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | string | The input date string |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | object | An object containing start_date as the input date and end_date as the next day in YYYY-MM-DD format |

### `secondsToISODate`
#### Description
Converts Unix timestamp (in seconds) to an ISO date string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| unixTime | number | Unix timestamp in seconds |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isoDate | string | The formatted ISO date string (YYYY-MM-DD) |

### `secondsToISODateTime`
#### Description
Converts Unix timestamp (in seconds) to an ISO date-time string.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| unixTime | number | Unix timestamp in seconds |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isoDateTime | string | The formatted ISO date-time string |

### `parseISODate`
#### Description
Parses and validates an ISO date string, throwing an error if invalid.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | string \| undefined | The input date string to parse |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| parsedDate | string \| undefined | The validated ISO date string or undefined |

#### Error Handling
Throws a `BadRequestError` if the input date is invalid.

### `isValidISODate`
#### Description
Checks if a given string is a valid ISO date.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | string | The input date string to validate |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isValid | boolean | True if the input is a valid ISO date, false otherwise |

#### Performance Considerations
This function is marked as deprecated. Consider using `@metriport/shared` instead for better performance and maintainability.
