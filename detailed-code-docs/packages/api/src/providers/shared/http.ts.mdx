## High-level description
This code defines a utility function `getHttpClient` that creates and returns an instance of Axios, a popular HTTP client for making API requests. The function allows for customization of the request timeout.

## Symbols

### `getHttpClient`
#### Description
This function creates and returns an Axios instance with a configurable timeout.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| timeoutInMs | number | Optional. The timeout in milliseconds for the HTTP requests. Defaults to `DEFAULT_TIMEOUT_MS`. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| AxiosInstance | AxiosInstance | An instance of Axios configured with the specified timeout. |

#### Internal Logic
1. The function uses `axios.create()` to create a new Axios instance.
2. It sets the `timeout` option of the Axios instance to the provided `timeoutInMs` or the default value.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| axios | Used to create the HTTP client instance |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| DEFAULT_TIMEOUT_MS | number | 20000 | The default timeout in milliseconds for HTTP requests |

## Performance Considerations
The `DEFAULT_TIMEOUT_MS` is set to 20 seconds, which may be suitable for most API calls. However, for specific endpoints that might take longer to respond, you may want to pass a custom timeout value when calling `getHttpClient`.
