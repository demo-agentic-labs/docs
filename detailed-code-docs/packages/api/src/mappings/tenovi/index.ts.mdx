## High-level description
The code defines TypeScript types for Tenovi measurement data using Zod schemas. These schemas define the structure and data types for individual Tenovi measurements and arrays of measurements.

## Code Structure
The code defines two Zod schemas: `tenoviMeasurementSchema` for individual measurements and `tenoviMeasurementDataSchema` for arrays of measurements. It also defines corresponding TypeScript types using `z.infer`.

## Symbols
### `tenoviMeasurementSchema`
#### Description
This Zod schema defines the structure and data types for a single Tenovi measurement.

#### Inputs
None. This is a Zod schema definition, not a function.

#### Outputs
None. This is a Zod schema definition, not a function.

#### Internal Logic
The schema defines the following fields:
- `metric`: A string representing the type of measurement (e.g., "weight", "blood_pressure").
- `device_name`: A string representing the name of the device.
- `hwi_device_id`: A string representing the hardware ID of the device.
- `patient_id`: A string representing the ID of the patient.
- `hardware_uuid`: A string representing the UUID of the hardware.
- `sensor_code`: A string representing the code of the sensor.
- `value_1`: A string or number representing the primary measurement value.
- `value_2`: A nullable string or number representing a secondary measurement value (e.g., diastolic blood pressure).
- `created`: A string representing the creation timestamp of the measurement.
- `timestamp`: A string representing the timestamp of the measurement.
- `timezone_offset`: A number representing the timezone offset.
- `estimated_timestamp`: A boolean indicating whether the timestamp is estimated.

### `tenoviMeasurementDataSchema`
#### Description
This Zod schema defines the structure and data types for an array of Tenovi measurements.

#### Inputs
None. This is a Zod schema definition, not a function.

#### Outputs
None. This is a Zod schema definition, not a function.

#### Internal Logic
The schema defines an array of objects conforming to the `tenoviMeasurementSchema`.
