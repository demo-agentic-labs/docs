## High-level description
This code defines a TypeScript interface named `WhoopBody` for representing body metrics data retrieved from the Whoop API. It uses the `zod` library for schema validation and type inference.

#### Internal Logic
Defines the expected shape of the Whoop API response using `zod` object schema. It specifies the following fields:
- `height_meter`: A number representing height in meters.
- `weight_kilogram`: A number representing weight in kilograms.
- `max_heart_rate`: A number representing the maximum heart rate.
