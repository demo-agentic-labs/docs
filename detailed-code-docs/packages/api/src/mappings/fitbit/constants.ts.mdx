## High-level description
This file defines constants and types related to the Fitbit data provider integration. It includes constants for metric units, locales, Fitbit collection types, and Fitbit API scopes.

## Code Structure
This file primarily defines constants and types that are used across different parts of the Fitbit integration, such as webhook processing and data mapping.
