## High-level description
The code defines a Zod schema, `fitbitTempSkinResp`, to validate and type-check data fetched from the Fitbit API, specifically the temperature skin summary endpoint. It also defines a corresponding TypeScript type, `FitbitTempSkin`, based on the schema.

## References
- `z` from the "zod" library.

## Symbols

### `fitbitTempSkinResp`
#### Description
A Zod schema that validates the response from the Fitbit API's temperature skin summary endpoint.

#### Inputs
None. It's a Zod schema definition, not a function.

#### Outputs
None. It's a Zod schema definition, not a function.

#### Internal Logic
The schema defines the expected structure and data types for the Fitbit temperature skin summary response:
- `dateTime`: A required string representing the date and time.
- `value`: A required object containing:
    - `nightlyRelative`: An optional number representing the nightly relative temperature.
- `logType`: A required string representing the log type.

### `FitbitTempSkin`
#### Description

#### Outputs
