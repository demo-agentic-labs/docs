## High-level description
This code defines a TypeScript type, `FitbitTempCore`, to represent Fitbit temperature core data. It uses the `zod` library for validation and type inference, ensuring the data structure aligns with the Fitbit API response. 

## References
- This code references the `fitbitTempCoreResp` symbol, which is a Zod schema defined in the same file.

## Symbols
### `fitbitTempCoreResp`
#### Description
This symbol is a Zod schema that defines the expected structure of a Fitbit temperature core response from the  `/temp/core/date/{date}.json` endpoint. It validates that the response is an optional object containing `dateTime` (string) and `value` (optional number) properties.

#### Inputs
*This symbol does not accept any inputs.*

#### Outputs
*This symbol does not return any outputs. It is used for type validation and inference.*

### `FitbitTempCore`
#### Description
This symbol is a TypeScript type inferred from the `fitbitTempCoreResp` Zod schema. It represents the validated data structure of a Fitbit temperature core response.

#### Inputs
*This symbol does not accept any inputs.*

#### Outputs
*This symbol does not return any outputs. It is a type definition.*

## Dependencies
- **zod:** Used for schema validation and type inference.
