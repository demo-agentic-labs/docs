## High-level description
This code defines types and validation schemas related to Google Fit body data, such as weight, height, and body fat percentage. It uses the `zod` library for schema validation and defines constants for Google Fit data source IDs.

## Code Structure
The code defines several constants for Google Fit data source IDs and uses them in a Zod schema (`googleBodyResp`) to validate the response from the Google Fit API. The `googleBodyResp` schema is created using a helper function `googleResp` imported from another file. The code also exports a type alias `GoogleBody` for the validated Google Fit body data.

## References
- `googleResp`: This symbol is imported from another file (`./index.ts`) and is used to define the `googleBodyResp` schema.
