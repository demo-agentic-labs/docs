## High-level description
This code defines types and constants related to Google Fit activity data, including data source IDs, activity types, and response models. It provides a mapping between Google Fit's activity data structure and the standardized format used within the application.

## Code Structure
The code defines several constants for Google Fit data source IDs and a mapping of activity type codes to human-readable names. It utilizes Zod schemas to define the expected structure of Google Fit API responses. These schemas are used to validate and type-check the data received from the Google Fit API.
