## High-level description
This file contains test utility functions for creating mock Facility objects and related data structures. It's primarily used for testing purposes in the Metriport API, specifically for medical facility-related functionality.

## Code Structure
The file defines several functions that create mock data objects, including `makeFacilityNumber`, `makeFacilityData`, `makeFacility`, and `makeFacilityModel`. These functions are interrelated, with `makeFacility` using `makeFacilityData` and `makeFacilityModel` using `makeFacility`.

## Symbols

### `makeFacilityNumber`
#### Description
Generates a random facility number between 0 and 1,000,000.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| number | number | A random integer between 0 and 1,000,000 |
