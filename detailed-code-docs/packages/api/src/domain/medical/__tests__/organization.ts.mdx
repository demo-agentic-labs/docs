## High-level description
This file contains test utility functions for creating mock Organization and OrganizationData objects. It's primarily used for testing purposes in the Metriport API, specifically for medical-related functionalities.

## Code Structure
The file exports three main functions: `makeOrgNumber`, `makeOrganizationData`, and `makeOrganization`. These functions use the Faker library to generate random data for creating mock organization objects.

## Symbols

### `makeOrgNumber`
#### Description
Generates a random organization number.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | number | A random integer between 0 and 1,000,000 |

### `makeOrganizationData`
#### Description
Creates a mock OrganizationData object with random values.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | OrganizationData | An object containing name, type, and location |

### `makeOrganization`
#### Description
Creates a mock Organization object with random values, optionally overriding specific fields.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;Organization&gt; | Optional partial Organization object to override default values |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | Organization | A complete Organization object |

#### Internal Logic
1. Spreads the base domain properties using `makeBaseDomain`
2. Generates random values for required fields if not provided in the input
3. Uses `makeOrganizationData` to generate the `data` field
4. Sets default values for boolean fields
