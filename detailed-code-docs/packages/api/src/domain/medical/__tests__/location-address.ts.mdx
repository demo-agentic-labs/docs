## High-level description
This code defines two functions, `makeAddressStrict` and `makeAddressWithCoordinates`, which are used to generate mock address data for testing purposes. These functions utilize the Faker library to create realistic, randomized address information.

## Symbols

### `makeAddressStrict`
#### Description
This function creates a mock `AddressStrict` object with randomized address details.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;AddressStrict&gt; | Optional partial AddressStrict object to override default values |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| AddressStrict | object | A complete AddressStrict object with randomized or provided values |

#### Internal Logic
1. Generates a random index to select a US state.
2. Creates an object with address properties, using provided params or generating random values using Faker.
3. Returns the created AddressStrict object.

### `makeAddressWithCoordinates`
#### Description
This function creates a mock `AddressWithCoordinates` object, which extends `AddressStrict` with latitude and longitude coordinates.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;AddressWithCoordinates&gt; | Optional partial AddressWithCoordinates object to override default values |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| AddressWithCoordinates | object | A complete AddressWithCoordinates object with randomized or provided values |

#### Internal Logic
1. Calls `makeAddressStrict` to generate the base address information.
2. Adds `coordinates` property with `lat` and `lon` values, using provided params or generating random values using Faker.
3. Returns the created AddressWithCoordinates object.
