## High-level description
This file contains unit tests for the `queryToSearchParams` function, which converts query parameters into URLSearchParams. The tests cover various scenarios, including empty inputs, simple string parameters, and arrays of parameters.

## Symbols

### `describe("queryToSearchParams", () =&gt; { ... })`
#### Description
This is the main test suite for the `queryToSearchParams` function. It contains multiple test cases to verify the function's behavior under different input conditions.

### `it("returns empty when gets empty", () =&gt; { ... })`
#### Description
This test case verifies that the function returns an empty URLSearchParams object when given an empty input.

#### Internal Logic
1. Call `queryToSearchParams` with an empty object.
2. Assert that the result is an instance of URLSearchParams.
3. Assert that the result is equal to a new, empty URLSearchParams object.
