## High-level description
This code defines Zod schemas for validating and structuring organization-related data in a medical API. It includes schemas for creating and updating organizations, as well as for internal organization details, utilizing enums and address schemas from related files.

## Code Structure
The code defines several Zod schemas that are related to each other. The `organizationCreateSchema` and `organizationUpdateSchema` are identical and use the `orgTypeSchema` and `AddressStrictSchema`. The `organiationInternalDetailsSchema` extends the `AddressStrictSchema` with additional fields.

## References
This code references types and schemas from other parts of the codebase, including:
- `OrgType` and `OrganizationBizType` from `@metriport/core/domain/organization`
- `AddressStrictSchema` from `./address`
