## High-level description
This code defines schemas and functions for handling patient data in a medical API. It includes functionality for creating, updating, and converting patient data between different formats, as well as validating and normalizing various patient-related fields.

## Code Structure
The code defines several schemas using Zod for data validation and transformation. It also exports functions that convert between different patient data representations, such as from API schemas to internal domain models.
