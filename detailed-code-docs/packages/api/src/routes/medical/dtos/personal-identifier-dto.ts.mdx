## High-level description
This code defines a type alias `PersonalIdentifierDTO` that is equivalent to the `PersonalIdentifier` type imported from the `@metriport/core/domain/patient` module. It serves as a Data Transfer Object (DTO) for personal identifier information in the medical routes of the API.

## Symbols

### PersonalIdentifierDTO
#### Description
`PersonalIdentifierDTO` is a type alias that directly maps to the `PersonalIdentifier` type. It is used to represent personal identifier information in a format suitable for data transfer in the API.

#### Type Definition
```typescript
export type PersonalIdentifierDTO = PersonalIdentifier;
```

The `PersonalIdentifier` type, as seen in the related code snippet, is defined as:

```typescript
export type PersonalIdentifier = BaseIdentifier &
  (
    | { type: GeneralPersonalIdentifiers }
    | { type: DriversLicensePersonalIdentifier; state: USState }
  );
```

Where `BaseIdentifier` is:

```typescript
export type BaseIdentifier = {
  value: string;
  period?: Period;
  assigner?: string;
};
```

And `Period` is:

```typescript
export type Period =
  | {
      start: string;
      end?: string;
    }
  | {
      start?: string;
      end: string;
    };
```

The `PersonalIdentifier` type represents either a general personal identifier or a driver's license, with additional properties based on the type.
