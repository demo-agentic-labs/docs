## High-level description
This code defines Data Transfer Objects (DTOs) for organizations in a medical API. It provides structures for both public and internal organization representations, along with functions to convert organization models to these DTOs.

## Code Structure
The code defines two main DTO types: `OrganizationDTO` for public use and `InternalOrganizationDTO` for internal use. It also includes two functions, `dtoFromModel` and `internalDtoFromModel`, which convert an `Organization` model to the respective DTO formats.

### `dtoFromModel`
#### Description
Converts an `Organization` model to an `OrganizationDTO`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| org | Organization | The organization model to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | OrganizationDTO | The converted DTO |

#### Internal Logic
1. Extracts `name`, `type`, and `location` from `org.data`
2. Uses `toBaseDTO` to get base DTO fields
3. Combines extracted data with base DTO and `org.oid` to create the DTO
