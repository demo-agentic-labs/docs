## High-level description
The `removeDocRefMapping` function is a simple utility function that removes a document reference mapping from the database. It takes the `cxId` and `docRefMappingId` as input and returns the number of rows deleted.

## References
- This function is called from the `triggerDownloadDocument` function in `packages/api/src/external/commonwell/document/document-query.ts` file.

## Symbols

### `removeDocRefMapping`
#### Description
This function removes a document reference mapping from the database.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cxId | string | The CX ID. |
| docRefMappingId | string | The document reference mapping ID. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
|  | number | The number of rows deleted. |

#### Internal Logic
1. Constructs a `docRef` object with the `cxId` and `docRefMappingId`.
2. Calls the `destroy` method of the `DocRefMappingModel` to delete the mapping from the database.
3. Returns the result of the `destroy` method, which is the number of rows deleted.
