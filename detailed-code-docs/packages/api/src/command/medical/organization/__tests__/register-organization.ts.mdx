## High-level description
This code defines test data representing a medical organization's address with coordinates. It utilizes the Faker.js library to generate realistic-looking data for testing purposes. 

## References
- `@metriport/shared` for `USState` enum.
- `@metriport/core/domain/address` for `Coordinates` and `AddressWithCoordinates` types.

## Symbols

### `coordinates`
#### Description
A constant variable of type `Coordinates` representing geographical coordinates.

#### Inputs
_N/A_ This is a constant variable.

#### Outputs
_N/A_ This is a constant variable.

#### Internal Logic
- `lat`: Latitude generated using `faker.location.latitude()`.
- `lon`: Longitude generated using `faker.location.longitude()`.

### `addressWithCoordinates`
#### Description
A constant variable of type `AddressWithCoordinates` representing a complete address with geographical coordinates.

#### Inputs
_N/A_ This is a constant variable.

#### Outputs
_N/A_ This is a constant variable.

#### Internal Logic
- Utilizes `faker.js` to generate realistic address data:
    - `addressLine1`: Generated using `faker.location.streetAddress()`.
    - `addressLine2`: Generated using `faker.location.secondaryAddress()`.
    - `city`: Generated using `faker.location.city()`.
    - `state`: Set to `USState.CA`.
    - `zip`: Generated using `faker.location.zipCode()`.
    - `country`: Set to `"USA"`.
    - `coordinates`: Uses the previously defined `coordinates` constant.
