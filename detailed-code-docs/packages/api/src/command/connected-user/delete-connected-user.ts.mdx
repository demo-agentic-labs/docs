## High-level description
The `deleteConnectedUser` function is a database operation that removes a user's connection data from the system. It takes the user's ID as input and deletes the corresponding record from the `ConnectedUser` table.

## References
- `ConnectedUser`: This symbol refers to the `ConnectedUser` model, which represents the database table storing user connection information.

## Symbols

### `deleteConnectedUser`
#### Description
This function deletes a connected user's record from the database based on their ID.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| userId | string | The ID of the connected user to be deleted. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | Promise\&lt;void\&gt; | A promise that resolves when the deletion is complete. |

#### Internal Logic
1. Uses the `ConnectedUser.destroy` method to delete the connected user record.
2. The `where` clause specifies that the record with the matching `userId` should be deleted.

## Side Effects
This function modifies the database by deleting a connected user record.
