## High-level description
The `isDAPIWebhookRequest` function determines if a given webhook request originated from the DAPI (Devices API) system. It achieves this by checking if the request type string starts with the DAPI webhook prefix.

## References
- `dapiWHPrefix`: A constant representing the prefix used for DAPI webhook types.
- `WebhookRequest`: A model representing a webhook request.
- `WebhookRequestData`: A type representing data for a webhook request.

## Symbols

### `isDAPIWebhookRequest`
#### Description
This function checks if a webhook request originated from the DAPI system.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhookRequest | `WebhookRequest` \| `WebhookRequestData` | The webhook request object to check. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | `boolean` | True if the request is a DAPI webhook request, false otherwise. |

#### Internal Logic
The function simply checks if the `type` property of the `webhookRequest` starts with the `dapiWHPrefix`. If it does, it returns `true`, otherwise `false`.
```typescript
return webhookRequest.type.startsWith(dapiWHPrefix);
```
