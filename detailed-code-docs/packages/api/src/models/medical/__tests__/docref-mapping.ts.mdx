### `makeDocRefMappingModel`
#### Description
This function is a test utility that creates and returns an instance of `DocRefMappingModel` based on the provided parameters. It leverages the `makeDocRefMapping` function to generate the `DocRefMapping` object and then casts it to `DocRefMappingModel`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | `Partial&lt;DocRefMapping&gt;` | (Optional) An object containing partial properties to override the default values of a `DocRefMapping` object. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | `DocRefMappingModel` | An instance of `DocRefMappingModel` populated with the provided parameters or default values. |

#### Internal Logic
1. Calls `makeDocRefMapping(params)` to create a `DocRefMapping` object with the given parameters.
2. Casts the returned `DocRefMapping` object to `DocRefMappingModel`.
3. Returns the created `DocRefMappingModel` instance.
