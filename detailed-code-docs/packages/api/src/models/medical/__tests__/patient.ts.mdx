## High-level description
The `makePatientModel` function is a test utility function that creates and returns an instance of the `PatientModel` class, populated with data from a `Patient` object. This function is used in tests to easily create mock `PatientModel` instances for testing purposes.

## Symbols

### `makePatientModel`
#### Description
Creates and returns an instance of `PatientModel` populated with data from a `Patient` object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | `Partial&lt;PatientModel&gt;` | Optional. Partial `PatientModel` object to override default values. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| model | `PatientModel` | An instance of `PatientModel` populated with data. |

#### Internal Logic
1. Creates a `Patient` object using the `makePatient` function, optionally overriding default values with the provided `params`.
2. Creates a new instance of `PatientModel` using the created `Patient` object.
3. Assigns the `data` property of the `model` to the `data` property of the created `Patient` object.
4. Returns the created and populated `PatientModel` instance.
