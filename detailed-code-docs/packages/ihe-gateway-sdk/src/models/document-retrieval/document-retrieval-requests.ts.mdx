## High-level description
This code defines schemas and types for document retrieval requests in an IHE (Integrating the Healthcare Enterprise) gateway system. It includes both inbound and outbound request schemas, which are used to validate and structure data for document retrieval operations in healthcare information exchange.

## Code Structure
The code defines two main schemas:
1. `outboundDocumentRetrievalReqSchema`: For outbound document retrieval requests.
2. `inboundDocumentRetrievalReqSchema`: For inbound document retrieval requests.

These schemas extend a base schema (`documentRetrievalReqDefaultSchema`) and add specific fields for their respective use cases.
