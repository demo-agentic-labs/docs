## High-level description
This code defines schemas and types for document query requests in an IHE (Integrating the Healthcare Enterprise) gateway SDK. It includes both inbound and outbound request schemas, as well as utility types and constants related to date ranges.

## Code Structure
The code defines several schemas using the Zod library:
- `dateRangeSchema`: Defines a date range with `dateFrom` and `dateTo` properties.
- `documentQueryDefaultReqSchema`: Extends a base request schema with additional properties specific to document queries.
- `outboundDocumentQueryReqSchema`: Extends the default schema for outbound requests.
- `inboundDocumentQueryReqSchema`: Uses the default schema for inbound requests.

## Symbols

### `dateRangeSchema`
#### Description
Defines a schema for a date range with two string properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| dateFrom | string | The start date of the range |
| dateTo | string | The end date of the range |

### `DateRange`
#### Description
A type alias for the inferred type of `dateRangeSchema`.

### `documentQueryDefaultReqSchema`
#### Description
Extends the base request schema with additional optional properties specific to document queries.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| externalGatewayPatient | object | Patient information for the external gateway |
| classCode | object (optional) | Classification code for the document |
| practiceSettingCode | object (optional) | Practice setting code |
| facilityTypeCode | object (optional) | Facility type code |
| documentCreationDate | DateRange (optional) | Date range for document creation |
| serviceDate | DateRange (optional) | Date range for service |

### `outboundDocumentQueryReqSchema`
#### Description
Extends the default document query schema for outbound requests.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| gateway | object | Information about the XCA gateway |
| patientId | string | Patient identifier |
| cxId | string | Context identifier |

### `OutboundDocumentQueryReq`
#### Description
A type alias for the inferred type of `outboundDocumentQueryReqSchema`.

### `inboundDocumentQueryReqSchema`
#### Description
Uses the default document query schema for inbound requests.

### `InboundDocumentQueryReq`
#### Description
A type alias for the inferred type of `inboundDocumentQueryReqSchema`.

## Dependencies
- `zod`: Used for schema definition and validation.
- `../shared`: Imports shared schemas and types.

This code provides a structured way to define and validate document query requests in both inbound and outbound scenarios, ensuring consistency and type safety in the IHE gateway SDK.
