## High-level description
This file is an ESLint configuration file for the ihe-gateway-sdk package. It sets up the linting rules and parser for TypeScript projects, extending recommended configurations and integrating with Prettier for code formatting.

## Symbols

### module.exports
#### Description
This is the main configuration object exported by the module. It defines the ESLint settings for the project.

#### Inputs
N/A (This is a configuration object, not a function)

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| module.exports | Object | ESLint configuration object |

#### Internal Logic
The configuration object contains the following properties:

1. `extends`: An array of configurations to extend from.
2. `parser`: Specifies the parser to be used for linting.
3. `plugins`: An array of plugins to be used.
4. `root`: A boolean indicating this is the root configuration file.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| eslint:recommended | Provides a set of recommended ESLint rules |
| plugin:@typescript-eslint/recommended | Provides recommended TypeScript-specific linting rules |
| prettier | Integrates Prettier formatting with ESLint |
| @typescript-eslint/parser | Allows ESLint to parse TypeScript code |
| @typescript-eslint | Plugin that provides TypeScript-specific linting rules |
