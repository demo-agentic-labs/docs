## High-level description
This code defines a TypeScript type `ConnectedUserInfo` which represents information about a connected user in the Metriport API. It includes the user's Metriport ID, application-specific user ID, and an optional array of connected providers.

## Symbols

### `ConnectedUserInfo`
#### Description
`ConnectedUserInfo` is a TypeScript type that defines the structure for connected user information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| metriportUserId | string | The unique identifier for the user in Metriport's system |
| appUserId | string | The unique identifier for the user in the client application |
| connectedProviders | string[] | Optional. An array of strings representing the providers connected to the user |
