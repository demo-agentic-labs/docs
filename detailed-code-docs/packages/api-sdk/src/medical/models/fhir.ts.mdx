## High-level description
This file defines various types, schemas, and exports related to the FHIR (Fast Healthcare Interoperability Resources) data model used in the Metriport Medical API. It includes definitions for resource types, consolidation filters, and query-related structures.

## Symbols

#### Description
An array of FHIR resource types that can be searched using a patient reference.

#### Description
A type representing the possible values of `resourcesSearchableByPatient`.
