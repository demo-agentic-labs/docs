## High-level description
This code defines a utility function `makeContact` for creating test data for the `Contact` type. It uses the Faker library to generate realistic mock data for phone numbers and email addresses.

## Symbols

### `makeContact`
#### Description
The `makeContact` function is a factory function that creates a `Contact` object with optional overrides. It generates random phone numbers and email addresses using Faker when not provided.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| params | Partial&lt;Contact&gt; | Optional. Partial Contact object to override default generated values. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| contact | Contact | A Contact object with phone and email fields. |

#### Internal Logic
1. The function uses the spread operator to merge the provided `params` with default values.
2. If `phone` is not provided in `params`, it generates a random 10-digit phone number using Faker.
3. If `email` is not provided in `params`, it generates a random email address using Faker.
