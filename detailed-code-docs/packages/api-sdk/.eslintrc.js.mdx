## High-level description
This file, `.eslintrc.js`, is the ESLint configuration file for the `@metriport/api-sdk` package. It sets up the linting rules and parser for TypeScript files in the project, ensuring code quality and consistency.

## Code Structure
The file exports a single configuration object that defines the ESLint settings for the project.

## Symbols

### module.exports
#### Description
This is the main configuration object exported by the file. It defines the ESLint settings for the project, including extended configurations, parser, plugins, and root settings.

#### Inputs
N/A

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| module.exports | Object | ESLint configuration object |

#### Internal Logic
The configuration object includes the following properties:
1. `extends`: An array of ESLint configurations to extend from.
2. `parser`: Specifies the parser to be used for TypeScript files.
3. `plugins`: An array of ESLint plugins to be used.
4. `root`: A boolean indicating that this is the root configuration file.

## Dependencies
The configuration relies on the following ESLint plugins and configurations:
| Dependency | Purpose |
|:-----------|:--------|
| eslint:recommended | Provides a set of recommended ESLint rules |
| plugin:@typescript-eslint/recommended | Provides recommended rules for TypeScript |
| prettier | Integrates Prettier formatting with ESLint |
| @typescript-eslint/parser | Allows ESLint to parse TypeScript files |
| @typescript-eslint | Provides TypeScript-specific linting rules |

## Configuration
The ESLint configuration is set up with the following options:
| Option | Type | Value | Description |
|:-------|:-----|:------|:------------|
| extends | Array | ['eslint:recommended', 'plugin:@typescript-eslint/recommended', 'prettier'] | Extends from recommended ESLint, TypeScript, and Prettier configurations |
| parser | String | '@typescript-eslint/parser' | Uses the TypeScript parser for ESLint |
| plugins | Array | ['@typescript-eslint'] | Includes the TypeScript ESLint plugin |
| root | Boolean | true | Indicates this is the root configuration file |

This configuration ensures that the project follows recommended JavaScript and TypeScript linting rules while also integrating with Prettier for consistent code formatting. The use of `@typescript-eslint` parser and plugin allows for TypeScript-specific linting, improving code quality for TypeScript files in the project.
