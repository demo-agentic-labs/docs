## High-level description
This file, `tsconfig.ref.json`, is a TypeScript configuration file specifically designed for the `tester-node` package within the TypeScript SDK. It extends the base `tsconfig.json` configuration and adds specific settings for creating a composite project.

## Code Structure
The configuration file consists of a single JSON object with three main properties: `extends`, `include`, and `compilerOptions`.

## Symbols

### Root Configuration Object
#### Description
The root object defines the TypeScript compiler options and project structure for the `tester-node` package.

#### Internal Logic
1. Extends the base configuration from `./tsconfig.json`
2. Specifies files to include in the compilation
3. Sets compiler options specific to this project

### `extends`
#### Description
This property specifies that this configuration file extends another TypeScript configuration file.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| extends | string | Path to the base configuration file |

### `include`
#### Description
This property specifies which files should be included in the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| include | array of strings | Glob patterns for files to include |

### `compilerOptions`
#### Description
This property sets specific compiler options for the TypeScript compiler.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| composite | boolean | Enables TypeScript's project references feature |

## Dependencies
This configuration file depends on the existence of a `tsconfig.json` file in the same directory, which it extends.

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| extends | string | "./tsconfig.json" | Specifies the base configuration file to extend |
| include | array | ["src/**/*"] | Glob pattern for files to include in compilation |
| composite | boolean | true | Enables TypeScript's project references feature |
