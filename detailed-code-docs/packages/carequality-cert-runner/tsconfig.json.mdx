## High-level description
This `tsconfig.json` file configures TypeScript compiler options for the `carequality-cert-runner` package. It extends recommended TypeScript configurations and sets strict compilation rules to ensure code quality and type safety.

## Code Structure
The configuration is organized into three main sections: `extends`, `compilerOptions`, and file inclusion/exclusion rules.

## Symbols

### compilerOptions
#### Description
This object defines various TypeScript compiler options to control the compilation process and enforce coding standards.

#### Internal Logic
The compiler options are set to:
1. Target ECMAScript 2020 (`"target": "es2020"`)
2. Generate source maps (`"sourceMap": true`)
3. Emit declaration files (`"declaration": true`, `"declarationMap": true`)
4. Enable decorator metadata and experimental decorators
5. Enforce strict coding rules (from @tsconfig/strictest)
6. Set output directory to "./dist"
7. Enable additional strict checks for indexed access and property access

### include
#### Description
Specifies which files to include in the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| "./src" | string | The source directory to include |

### exclude
#### Description
Specifies which files or directories to exclude from the compilation process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| "dist" | string | The distribution directory to exclude |
| "node_modules" | string | The node_modules directory to exclude |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @tsconfig/recommended | Provides a base recommended TypeScript configuration |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| extends | string | "@tsconfig/recommended/tsconfig.json" | Extends the recommended TypeScript configuration |
| target | string | "es2020" | Specifies the ECMAScript target version |
| sourceMap | boolean | true | Generates corresponding .map files |
| declaration | boolean | true | Generates corresponding .d.ts files |
| declarationMap | boolean | true | Generates a sourcemap for .d.ts files |
| emitDecoratorMetadata | boolean | true | Enables experimental support for emitting type metadata for decorators |
| experimentalDecorators | boolean | true | Enables experimental support for decorators |
| outDir | string | "./dist" | Specifies the output directory for compiled files |
| noUncheckedIndexedAccess | boolean | true | Adds undefined to any un-declared field in an index signature |
| noPropertyAccessFromIndexSignature | boolean | true | Disallows property access from index signatures |

The configuration also includes several strict type-checking options from @tsconfig/strictest, such as `allowUnusedLabels: false`, `noFallthroughCasesInSwitch: true`, and `exactOptionalPropertyTypes: true`.
