---
title: "Get Settings"
description: "Gets the settings for your account."
api: "GET /settings"
---

## Response

&lt;ResponseField name="id" type="string" required&gt;
  The settings ID.
&lt;/ResponseField&gt;

&lt;ResponseField name="webhookUrl" type="string" required&gt;
  Your application's webhook URL.
&lt;/ResponseField&gt;

&lt;ResponseField name="webhookKey" type="string" required&gt;
  The key used to authenticate webhook calls to your app.
&lt;/ResponseField&gt;

&lt;ResponseExample&gt;

```javascript Metriport SDK
import { MetriportMedicalApi } from "@metriport/api-sdk";

const metriportClient = new MetriportMedicalApi("YOUR_API_KEY");

const response = await metriportClient.getSettings();
```

&lt;/ResponseExample&gt;

Example response:

```json
{
  "id": "&lt;settings-id&gt;",
  "webhookUrl": "https://server.domain.com/webhook/endpoint",
  "webhookKey": "&lt;webhook-key&gt;"
}
```
