---
title: "Update Settings"
description: "Updates your account's settings."
api: "POST /settings"
---

## Body

&lt;ParamField body="webhookUrl" type="string" required&gt;
  The webhook URL to be stored with your settings. It should be a valid URL.

  &lt;Info&gt;
    For requirements for this URL, see [our Webhooks guide](/home/api-info/webhooks).
  &lt;/Info&gt;
&lt;/ParamField&gt;

## Response

&lt;ResponseField name="id" type="string" required&gt;
  The settings ID.
&lt;/ResponseField&gt;

&lt;ResponseField name="webhookUrl" type="string" required&gt;
  Your application's webhook URL.
&lt;/ResponseField&gt;

&lt;ResponseField name="webhookKey" type="string" required&gt;
  The key used to authenticate webhook calls to your app.
&lt;/ResponseField&gt;

&lt;ResponseExample&gt;

```javascript Metriport SDK
import { MetriportMedicalApi } from "@metriport/api-sdk";

const metriportClient = new MetriportMedicalApi("YOUR_API_KEY");

const response = await metriportClient.updateSettings("https://api.health.com/webhook");
```

&lt;/ResponseExample&gt;

Example response:

```json
{
  "id": "&lt;settings-id&gt;",
  "webhookUrl": "https://server.domain.com/webhook/endpoint",
  "webhookKey": "&lt;webhook-key&gt;"
}
```
