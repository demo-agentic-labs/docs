&lt;ResponseField name="id" type="string" required&gt;
  The ID assigned to this Facility. This ID will be used to uniquely identify this Facility in
  medical documents.
&lt;/ResponseField&gt;

&lt;ResponseField name="oid" type="string" required&gt;
  The OID assigned to this Facility.
&lt;/ResponseField&gt;

&lt;ResponseField name="name" type="string" required&gt;
  The name of your Facility. This is the actual address where a patient will receive care (can be
  virtual) - for example `Care Facility, LLC`.
&lt;/ResponseField&gt;

&lt;ResponseField name="npi" type="string" required&gt;
  The 10 digit National Provider Identifier (NPI) that will be used to make requests on behalf of
  the Facility.
&lt;/ResponseField&gt;

&lt;ResponseField name="address" type="Address" required&gt;
  &lt;Snippet file="address-response.mdx" /&gt;
&lt;/ResponseField&gt;

&lt;ResponseField name="tin" type="string"&gt;
  The Taxpayer Identification Number (TIN) of the Facility.
&lt;/ResponseField&gt;

&lt;ResponseField name="active" type="bool"&gt;
  Whether or not this Facility is currently active - this is usually `true`.
&lt;/ResponseField&gt;
