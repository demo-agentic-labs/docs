&lt;ParamField query="userId" type="string" required&gt;
  The Metriport user ID to create a session token for.
&lt;/ParamField&gt;

&lt;ParamField query="date" type="string" required&gt;
  Day for which to pull the data. Formatted `YYYY-MM-DD`.

  &lt;Info&gt;
    For simplicity, all queries assume the UTC timezone for the specified date
    starting at 00:00 to 23:59 - unless otherwise specified by the `timezoneId`
    parameter.
  &lt;/Info&gt;
&lt;/ParamField&gt;

&lt;ParamField query="timezoneId" type="string" optional&gt;
  Timezone ID to use for the specified date. If not specified, the UTC timezone is used.
  For example this can be valid timezones like `America/Los_Angeles` or `Europe/Kiev`.

  &lt;Info&gt;
    Valid timezones can be found at https://en.wikipedia.org/wiki/List_of_tz_database_time_zones.
  &lt;/Info&gt;

  &lt;Warning&gt;
    Support for this parameter is under development, and currently only supported
    for the Google Fit provider.
  &lt;/Warning&gt;
&lt;/ParamField&gt;
