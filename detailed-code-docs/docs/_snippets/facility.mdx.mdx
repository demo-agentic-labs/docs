&lt;ParamField body="name" type="string" required&gt;

  virtual) - for example `Care Facility, LLC`.
&lt;/ParamField&gt;

&lt;ParamField body="npi" type="string" required&gt;
  The 10 digit National Provider Identifier (NPI) that will be used to make requests on behalf of
  the Facility.
&lt;/ParamField&gt;

&lt;ParamField body="tin" type="string"&gt;

&lt;/ParamField&gt;
