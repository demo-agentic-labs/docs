&lt;Info&gt;
  You may provide a comma/space delimited string to specify multiple first and last names. For
  example, the following inputs would be equivalent: `"John,Jonathan"` & `"John Jonathan"` - these
  would translate to `["John", "Jonathan"]`.
&lt;/Info&gt;

&lt;ParamField body="firstName" type="string" required&gt;
  The Patient's first name(s).
&lt;/ParamField&gt;

&lt;ParamField body="lastName" type="string" required&gt;
  The Patient's last name(s).
&lt;/ParamField&gt;

&lt;ParamField body="dob" type="string" required&gt;
  The Patient's date of birth (DOB), formatted `YYYY-MM-DD` as per [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601).
&lt;/ParamField&gt;

&lt;ParamField body="genderAtBirth" type="string" required&gt;
  The Patient's gender at birth, can be one of `M` or `F` or `O` or `U`. Use `O` (other) when the patient's gender is known but it is not `M` or `F`, i.e intersex or hermaphroditic. 
  Use U (unknown) when the patient's gender is not known.
&lt;/ParamField&gt;

&lt;ParamField body="personalIdentifiers" type="PersonalIdentifier[]" optional&gt;
  An array of the Patient's personal IDs, such as a driver's license or SSN. May be empty.

  &lt;Expandable title="PersonalIdentifier properties"&gt;
    &lt;ParamField body="type" type="string" required&gt;
      The ID type - currently `driversLicense` or `ssn` is supported.
    &lt;/ParamField&gt;

    &lt;ParamField body="state" type="string" required&gt;
      The 2 letter state acronym where this ID was issued, for example: `CA`. Only required for type `driversLicense`.
    &lt;/ParamField&gt;

    &lt;ParamField body="value" type="string" required&gt;
      The ID number. For type `ssn`, should be 9 digits.
    &lt;/ParamField&gt;

  &lt;/Expandable&gt;
&lt;/ParamField&gt;

&lt;ParamField body="address" type="Address[]" required&gt;
  An array of `Address` objects, representing the Patient's current and/or previous addresses. May
  be empty.
  &lt;Snippet file="patient-address.mdx" /&gt;
&lt;/ParamField&gt;

&lt;ParamField body="contact" type="Contact[]"&gt;
  An array of `Contact` objects, representing the Patient's current and/or previous contact information. May be empty.
  &lt;Expandable title="Contact properties"&gt;
    &lt;ParamField body="phone" type="string"&gt;
      The Patient's 10 digit phone number, formatted `1234567899`.
    &lt;/ParamField&gt;

    &lt;ParamField body="email" type="string"&gt;
      The Patient's email address.
    &lt;/ParamField&gt;

  &lt;/Expandable&gt;

&lt;/ParamField&gt;
