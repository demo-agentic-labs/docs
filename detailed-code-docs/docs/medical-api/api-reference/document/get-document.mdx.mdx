---
title: "Get Document URL"
description: "Gets a downloadable URL for downloading the specified document."
api: "GET /medical/v1/document/download-url"
---

This endpoint returns a URL which you can use to download the specified document and/or convert using the file name provided from the [List Documents endpoint](/medical-api/api-reference/document/list-documents).

&lt;Info&gt;The URL will only be valid for 60 seconds before you need to request a new one.&lt;/Info&gt;

## Query Params

&lt;ParamField query="fileName" type="string" required&gt;
  The file name of the document.
&lt;/ParamField&gt;

&lt;ParamField query="conversionType" type="string"&gt;
  The document type to convert to. Either `html` or `pdf`.

  &lt;Info&gt;This parameter should only be used for converting XML/CDA files.&lt;/Info&gt;
&lt;/ParamField&gt;

## Response

A JSON object containing the URL will be returned.

&lt;ResponseField name="url" type="string"&gt;
  The downloadable URL.
&lt;/ResponseField&gt;

&lt;Tip&gt;
  For patients with a large volume documents, you can use the [Bulk Get Document URL](/medical-api/api-reference/document/download-url-bulk to get all download URLs, instead of doing it one at a time.
&lt;/Tip&gt;

&lt;ResponseExample&gt;

```javascript Metriport SDK
import { MetriportMedicalApi } from "@metriport/api-sdk";

const metriport = new MetriportMedicalApi("YOUR_API_KEY");

const downloadableUrl = await metriport.getDocumentUrl("abc123-def456");
```

&lt;/ResponseExample&gt;
