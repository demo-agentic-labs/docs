## High-level description
This directory contains documentation for the FHIR resources used in the medical API. Each file corresponds to a specific FHIR resource and provides a detailed description of its properties, search parameters, and inherited properties.

## What does it do?
This directory serves as a reference guide for developers working with the medical API. It provides detailed information about the structure and usage of each FHIR resource, allowing developers to understand how to interact with the API and build applications that leverage its functionality.
