### Scripts
#### Description
The `scripts` section defines various npm commands that can be run for the project. These scripts handle tasks such as installation, building, testing, linting, and deployment preparation.

#### Internal Logic
Key scripts include:
- `init`: Installs dependencies and runs a custom lambda script.
- `clean`: Cleans up build artifacts and node_modules.
- `prep-deploy`: Prepares the project for deployment by installing dependencies, building, testing, and preparing lambda functions.
- `build`: Builds all workspaces and lambda functions.
- `test`: Runs tests across all workspaces and lambda functions.
- `lambdas`: Executes a specified script within the lambdas package.
