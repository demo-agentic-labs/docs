## High-level description
This `tsconfig.ref.json` file is a TypeScript configuration file that extends the base `tsconfig.json` configuration. It specifically targets the source files in the `src` directory and enables composite project build mode, which is useful for structuring larger TypeScript projects.

## Code Structure
The file contains a single JSON object with three main properties: `extends`, `include`, and `compilerOptions`.

## Symbols

#### Description
The root object defines the TypeScript compiler options and project structure for this specific configuration.

#### Internal Logic
1. Extends the base `tsconfig.json` configuration.
2. Includes all files in the `src` directory and its subdirectories.
3. Sets specific compiler options for this configuration.

#### Description

#### Inputs
